半导体超级周期 — 第一性原理分析

First Principles Thinking · 非共识洞察 · 交易者视角

Date: 2026-03-19 Author: Cognitive Engine V3 Deep Think Type: 第一性原理分析

§0元问题：我们到底在分析什么？

思考起点

在看任何数据之前，先问三个元问题：

  1. 这个周期的本质是什么？ — 是技术革命、还是资本泡沫、还是两者叠加？
  2. 市场在定价什么故事？ — 当前估值隐含的增长假设是什么？
  3. 什么条件下这个故事会破裂？ — 不是"可能破裂"，而是"在什么精确条件下"？

思考过程记录

Q: 这轮半导体周期的本质到底是什么？

表面答案：AI需求驱动的超级周期。但这个答案太浅，因为每一轮科技周期都会被当时的人称为"超级周期"。

更深一层：这轮的独特之处不在于"AI"这个标签，而在于计算范式的转变。过去60年的计算遵循冯·诺依曼架构（CPU串行处理），AI推动的是向大规模并行计算的迁移。这不是一个应用层的变化，是基础设施层的变化。

类比思考：

  • 1990s互联网：从拨号→宽带，基础设施投资10年才见顶
  • 2000s移动互联网：从PC→手机，基础设施投资8年才见顶
  • 2020s AI：从CPU→GPU/ASIC，我们在第3年

第一性原理推论：如果这真的是计算范式转变（而非应用层泡沫），那基础设施投资周期应该持续5-10年，而非2-3年。我们在第3年，可能刚过中点。

但这里有一个关键假设需要验证：AI算力需求是否真的是无限的？

§1第一性原理分解

1.1 算力需求的第一性原理

关键洞察：Scaling Laws vs Efficiency

AI算力需求 = f(模型规模) × f(用户数) ÷ f(推理效率)

多数分析师只看前两项（越来越大的模型 × 越来越多的用户 = 无限需求）。但第三项——推理效率——被严重低估。

分解：

因子1: 模型规模。 GPT-4到GPT-5，参数量从~1.8T到可能10T+。训练算力需求确实在指数增长。但Scaling Law的回报正在递减——每10x算力投入，性能提升越来越小。这意味着训练侧的算力需求增速会在某个点减速（可能2027-2028）。

因子2: 用户数/推理量。 从ChatGPT到Agent、到嵌入每个应用。推理量可能是训练量的100-1000倍。这是真正的长期需求驱动力。但推理对算力的需求模式完全不同：推理侧更看性价比，不是绝对性能。这就是ASIC吃GPU增量蛋糕的结构性原因。

因子3: 效率提升。 2024-2025年的效率提升已经惊人：

  • 量化技术（FP16→INT8→INT4）：推理算力需求降低4-8倍
  • 稀疏化（MoE架构）：实际计算量降低3-5倍
  • 蒸馏（大模型→小模型）：同等任务算力降低10-50倍
  • 推理优化（KV-cache, Speculative decoding）：2-3倍加速

净效果：效率提升部分抵消了需求增长。但关键词是"部分"——因为需求增长是指数级的，而效率提升是线性的。在需求指数增长的早期，效率提升不重要；在后期，效率提升会成为增速杀手。

思考检查点

如果我的推论正确，那意味着：

  • 2024-2026: 需求指数增长 >> 效率提升 → 算力短缺（我们在这里）
  • 2027-2028: 需求增速放缓 + 效率提升积累 → 供需再平衡
  • 2029+: 推理侧成为主导 → 算力需求稳定增长，但不再是指数级

交易含义：这轮半导体超级周期的峰值可能在2027年前后，而非"永远涨"。

1.2 利润分配的第一性原理

问题：AI创造的价值，谁拿走了大头？

产业链利润分配遵循"微笑曲线"：

  • 上游（设计+IP）： NVDA毛利率75%，ARM royalty模式，SNPS/CDNS 70%+订阅续约率 — 拿走利润大头
  • 中游（制造）： TSMC毛利率63%，ASML 53% — 靠垄断拿走超额利润
  • 下游（应用）： 大多数AI应用公司亏损 — 几乎不赚钱

第一性原理推论：当下游（AI应用）持续亏损时，上游（芯片）的超额利润来源只有一个——Hyperscaler的资本支出。而Hyperscaler的capex来自两个来源：

  1. 自有现金流（稳定）
  2. 债务融资（$1,080亿累计负债，不稳定）
核心风险的第一性原理表述

半导体利润 = Hyperscaler Capex = f(AI应用ROI)

如果AI应用的ROI在2026-2027年无法验证（即用户不愿为AI付费），则：

AI应用亏损 → Hyperscaler削减capex → 芯片需求断崖 → 全链条利润崩塌

这是一个自指系统：芯片公司的利润来自还没有被证明能赚钱的AI应用。只要音乐在播放，大家继续跳舞；一旦有人问"ROI在哪？"，音乐可能停止。

1.3 垄断的第一性原理

为什么半导体利润率这么高？一个词：垄断。

六个节点中，有四个是事实上的垄断/寡头：

  • ASML: EUV 90%+ → 独家垄断（物理定律级壁垒：极紫外光源+超精密光学+复杂系统集成）
  • TSMC: 先进制程70% → 准垄断（20年工艺积累+客户迁移成本极高）
  • NVDA: GPU+CUDA 86% → 生态垄断（CUDA的20年开发者生态+数十亿沉没成本）
  • HBM三寡头: SK/Samsung/Micron = 100% → 寡头垄断（资本壁垒+技术壁垒+认证壁垒）

第一性原理推论：垄断的持久性取决于壁垒的性质：

  • 物理定律级壁垒（ASML）→ 几乎不可能被颠覆，除非出现全新的光刻技术（5-10年视野内不可能）
  • 工艺积累壁垒（TSMC）→ 可以被追赶但需要5-7年+数百亿美元（Samsung/Intel在尝试）
  • 生态壁垒（NVDA CUDA）→ 最容易被侵蚀，因为软件生态可以被新范式绕过（ASIC不需要CUDA）
  • 资本壁垒（HBM）→ 中等持久，新进入者（中国）可以通过国家资本突破

交易含义：按壁垒持久性排序的投资优先级应该是 ASML > TSMC > HBM > NVDA，但市场定价恰恰相反（NVDA市值最高）。这可能意味着NVDA是最大的估值错配风险。

§2非共识洞察：市场可能错在哪？

方法论

非共识 ≠ 反共识。不是为了反对而反对，而是寻找市场定价中隐含的假设，然后检验这些假设。

非共识洞察 #1: NVDA的CUDA护城河可能被高估

市场共识："CUDA是20年积累的不可逾越的壁垒"

第一性原理挑战：

  • CUDA是训练侧的壁垒，但AI正在从"训练为主"转向"推理为主"
  • 推理侧不需要CUDA的灵活性——固定模型、固定计算图、最优化到极致
  • 这正是ASIC的优势所在：Google TPU、Amazon Trainium就是为推理优化的
  • 2026年ASIC出货+44.6% vs GPU仅+16.1%——市场已经在用脚投票

推论：NVDA的份额从90%→75%不是"小幅下滑"，而是结构性转变的开始。训练侧NVDA仍然垄断（Scaling Law需要灵活性），但推理侧（占未来算力需求80%+）正在被ASIC蚕食。NVDA的长期TAM可能比市场预期小30-40%。

但反过来想：NVDA也在推出推理优化产品（Blackwell的推理性能4x提升），且Rubin架构2026 Q3专门强化推理。NVDA不会坐以待毙。所以这不是"NVDA完蛋了"，而是"NVDA的增速天花板比市场预期低"。

非共识洞察 #2: CoWoS瓶颈可能比芯片设计更值得关注

市场共识：半导体的价值在设计（NVDA/AVGO）和制造（TSMC）

第一性原理挑战：

  • 当前AI算力的实际瓶颈不是芯片设计能力，而是先进封装产能
  • CoWoS产能决定了GPU/ASIC的实际出货量——NVDA的Blackwell芯片设计完了，但能卖多少取决于TSMC的CoWoS产能
  • TSMC CoWoS从3.5万片/月→13万片/月是3.7倍扩产，但需求可能是10倍
  • 封装设备和材料供应商（日本的Disco、Ibiden、Shinko Electric等）可能是被低估的受益者

推论：投资半导体不应该只看"谁设计了最好的芯片"，还要看"谁控制了瓶颈"。封装环节可能是这轮周期中被最低估的环节。

非共识洞察 #3: 存储周期的"猪周期"风险被低估

市场共识："HBM需求结构性增长，供不应求持续到2027"

第一性原理挑战：

  • 存储行业有60年的周期历史，从未出现过"永远供不应求"
  • 三大厂（SK Hynix +维持、Samsung +50%、Micron +扩产）正在同时大幅扩产
  • "HBM售罄到2026底"与2018年初的"DRAM供不应求"叙事高度相似
  • 2018年DRAM周期见顶后：价格腰斩、SK Hynix股价-60%、Micron从$64→$28

但反过来想：HBM≠传统DRAM。HBM的技术壁垒更高（3D堆叠+TSV+CoWoS集成），新进入者更少，客户更集中（主要就是NVDA和AVGO）。所以这轮周期的振幅可能比传统DRAM周期小，但方向一定会重复——问题只是时间（2027还是2028见顶）。

非共识洞察 #4: 中国不是风险，是变量

市场共识："中国出口管制是利空"

第一性原理挑战：

  • 出口管制的短期效果：减少了收入（NVDA中国-40%、ASML中国-22pp）→ 利空
  • 出口管制的中期效果：逼迫客户提前囤货、加速非中国地区扩产 → 可能是净利好
  • 出口管制的长期效果：催生中国本土半导体产业链 → 远期竞争者，但现在还不是

推论：中国变量的净效果取决于时间框架。对于6-18个月的交易，中国因素可能是中性到微正（囤货需求+非中国扩产）。真正的风险不在中国，在于Hyperscaler的capex决策。

最佳类比判断

判断：最像1995-2000互联网，但有关键差异

相似之处：

  • 真实的技术革命 + 资本泡沫叠加
  • "卖铲子的先赚钱"（Cisco路由器 ≈ NVDA GPU）
  • 终端应用还没找到商业模式（1998年的电商 ≈ 2026年的AI Agent）
  • 基础设施过度投资（光纤过剩 ≈ 算力可能过剩）

关键差异：

  • 互联网时代没有垄断（Cisco份额<50%），AI时代有极端垄断（NVDA 86%）→ 崩盘时更集中
  • 互联网时代企业用自有资金投资，AI时代用债务融资（$1,080亿）→ 杠杆风险更大
  • 互联网时代估值更疯狂（PE 100-200x），AI时代相对理性（NVDA 30x forward）→ 泡沫程度可能更温和
  • 互联网的应用验证用了5年（Google 2004才盈利），AI的应用验证可能更快（Copilot/ChatGPT已有收入）

推论：这轮周期的上行空间可能不如1995-2000（因为估值已经比较理性），但下行风险也可能更温和（因为垄断者有真实利润支撑）。最可能的情景不是"Nasdaq -78%"式崩盘，而是"增速放缓+估值压缩"的软着陆——类似2022年科技股回调30-40%。

Cisco类比的警示

1999年的Cisco = 今天的NVDA？

  • Cisco 2000年市值$5,550亿（当时第一），NVDA 2026年市值$4.5万亿
  • Cisco的路由器是"必需品"但不是"独占品"——竞争者（Juniper、华为）最终侵蚀了份额
  • NVDA的GPU目前是"独占品"，但ASIC+AMD正在侵蚀推理侧
  • Cisco从高点跌了89%，至今（2026年）仍未回到2000年高点

但NVDA ≠ Cisco的关键原因：Cisco卖硬件（一次性），NVDA卖的是"CUDA生态+硬件+软件栈"（持续性）。NVDA的商业模式粘性更强。

结论：NVDA不会重演Cisco的-89%，但可能重演"增长天花板到来后的估值重估"——从30x forward PE压缩到15-20x，意味着股价可能从高点回调30-50%。

§4二阶效应：如果X发生，然后呢？

4.1 如果AI应用ROI被证实

一阶：Hyperscaler继续加码capex → 芯片需求持续

二阶：AI应用公司开始盈利 → 企业IT预算从传统软件转向AI → 传统软件公司（非AI）被挤压

三阶：AI成为效率工具 → 劳动力替代加速 → 宏观经济结构性变化（通缩压力+就业问题）

交易含义：做多半导体的同时，考虑做空传统IT服务公司（Accenture、Infosys等）作为对冲。

4.2 如果AI应用ROI被证伪

一阶：Hyperscaler削减capex → NVDA/TSMC收入下降

二阶：芯片库存积压 → 价格战 → HBM/DRAM暴跌（2018重演）

三阶：半导体公司削减研发 → 下一代技术（2nm、HBM4）延期 → 为下一轮周期底部播下种子

交易含义：如果2026 Q3-Q4看到capex指引下调，立即减仓HBM/存储标的（周期性最强），保留ASML/SNPS（垄断性最强）。

4.3 如果地缘政治升级

一阶：台海紧张 → TSMC估值折价

二阶：全球芯片供应链断裂 → 非TSMC代工厂（Samsung/Intel/GFS）反而受益

三阶：各国加速本土半导体建设 → ASML设备需求可能不降反升（更多工厂=更多设备需求）

交易含义：ASML可能是地缘政治风险的天然对冲——无论谁建厂，都需要ASML的设备。

4.4 如果效率提升超预期

一阶：同等任务需要更少算力 → 芯片需求放缓

二阶：但Jevons悖论——效率提升导致成本下降，成本下降导致使用量增加，净效果可能是需求不降反升

三阶：AI从高端应用扩散到边缘设备 → 终端AI芯片需求爆发（不是数据中心GPU）

交易含义：如果效率提升加速，关注边缘AI芯片（Qualcomm、MediaTek）而非数据中心GPU。

5.2 最关键的领先指标

如果只能看一个指标

Hyperscaler季度Capex指引的同比增速变化。

不是看绝对值（仍然很高），而是看增速的二阶导：

  • 增速加速（+36% → +40%）→ 周期仍在扩张，持有
  • 增速维持（+36% → +35%）→ 可能接近峰值，收紧止损
  • 增速减速（+36% → +20%）→ 周期拐点，减仓

时间窗口：2026年4-5月（Q1财报季）和7-8月（Q2财报季）是关键观察期。

5.3 矛盾信号

矛盾1：所有节点确认"需求强劲"，但Man Group/JPMorgan/BofA都在警告泡沫。
解读：短期现实（需求确实强劲）和中期风险（估值隐含的增长假设过于乐观）不矛盾。两者可以同时为真。

矛盾2：NVDA份额在下降，但收入在加速增长。
解读：蛋糕在快速膨胀，所以份额下降的同时绝对值仍在增长。但当蛋糕增速放缓时，份额下降就会转化为收入下降。这是一个延时炸弹。

矛盾3：存储"供不应求"，但三大厂在大幅扩产。
解读：经典周期行为——高利润吸引产能扩张，扩张导致过剩，过剩导致价格崩塌。问题只是时间。
