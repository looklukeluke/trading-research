🧠 28轮行业元思考的核心结论

  1. AI+半导体不是一个股票，是一条8节点供应链（含需求侧+AI Lab）。投资逻辑不是"买哪个"，而是"链上的权力在哪里、正在往哪里移动"
  2. 权力集中在两个chokepoint：TSMC(代工封装) + ASML(光刻设备)。20个代理人中，这两个节点的信号最先反映行业方向
  3. 20人的共识vs分歧才是投资地图。当Jensen+Wei+Lisa Su+Mehrotra+Altman+Nadella都说"需求insatiable"→ 95%可信（含买方验证）。当Jensen说"CUDA不可替代"但5个独立源矛盾→ 15%可信
  4. "不管谁赢都赚"的位置：AVGO(两条链交叉点) + TSM(所有芯片经过) + ASML(不关心谁的芯片)。NVDA只有bull case才赢大
  5. 最大风险不是技术失败，是共识拥挤。供给侧16人无一bearish，需求侧4人(Altman/Nadella/Pichai/Hassabis)新增但深度尚浅。当所有人都同意时最危险——这是2000年电信泡沫的共识结构
PHASE I — EXPLORE

我到底在做什么？

#1 表面任务 vs 真正任务

表面：整合20个认知代理人的洞察为一份行业研究。

真正：为趋势交易者回答一个问题——"AI+半导体这条链上，权力在哪里，正在往哪里移动，我该在哪个节点布局？"

这不是stock-picking报告，是产业链权力分析。个股只是权力位置的表达。

#2 素材盘点：我们有什么

380+文件，20个认知代理人（16供给侧+4需求侧），每人10+文件（Profile+V4素材地图+4份专项+元思考）。覆盖8个产业节点。

这可能是全球最详细的AI半导体认知画像数据库——不是财务数据，是关键决策者的认知模型。没有卖方研报有这个。

关键优势：我们不只知道"他们说了什么"，还知道"他们的偏见是什么"以及"他们说的和做的是否一致"。

#3 用户需要什么

趋势交易者的核心能力是"选品+周期匹配"。他需要：

  • 周期位置：AI+半导体大周期在哪？还有多少空间？
  • 品种筛选：链上7个节点，哪几个最有alpha？
  • 风险信号：什么时候该警惕？什么是kill switch？
  • 信号源分级：20个人谁最可信？怎么交叉验证？

#4 20人的共识：AI基建需求是真实的（含买方验证）

核心原则：Tier 1说的话直接用，Tier 2打折30%，Tier 3需要交叉验证，Tier 4只信方向不信数字。

PHASE III — DIAGNOSE

供应链权力地图

#7 权力集中度分析：谁不可替代？

核心发现：权力最集中的两个节点——ASML(100%EUV)和TSMC(90%+先进制程)——是整条链的"不可替代瓶颈"。所有其他节点都可以被替代（虽然替代很慢）。

投资含义：瓶颈节点有最强定价权。ASML和TSMC是"不管AI怎么发展，只要芯片在做就赚"的位置。

#8 权力正在往哪里移动？
权力上移：从芯片设计→基础设施 — AI客户(Google/Meta/Amazon)开始自研芯片→绕过NVIDIA→但仍需要TSMC代工+ASML光刻。权力从"谁设计芯片"向"谁能制造芯片"转移。
权力扩散：从NVIDIA独占→多方竞争 — AVGO帮客户自研+AMD竞争+Google TPU→NVIDIA份额从90%+向70-80%压缩(2027-2030)。但绝对需求增长可能抵消份额下降。
权力地理转移：台湾→美国/日本 — CHIPS Act+Arizona fab+熊本JASM。Morris Chang说"futile but my dream"。成本高30-50%，但地缘政治驱动不可逆。
#9 封装是新瓶颈

供给侧16个代理人的共识：先进封装(CoWoS)是2025-2027的真正瓶颈，不是晶圆制造。

TSMC的CoWoS产能从35K片/月→130K片/月(2026底)仍不够。NVIDIA锁定70%+ CoWoS产能。Vera Rubin的4x光罩设计需要+33%封装产能。

投资含义：封装设备供应商(AMAT的先进封装线、Kulicke & Soffa)是被低估的受益者。

PHASE IV — DIAGNOSE

AI+半导体周期在什么位置？

#10 周期位置判断

20个代理人的信号综合，当前周期位置：

需求侧：爆发期→高速增长期过渡 — FY2025 NVIDIA +114% → FY2026 +65% → FY2027指引~+40%。增速在下降，但绝对值仍在加速（每季度+$10B）。不是"泡沫破裂"，是"增速正常化"。
供给侧：瓶颈期 — TSMC CoWoS满载、SK hynix HBM供不应求、ASML交期18个月。瓶颈=定价权=利润率高。当瓶颈解除时=定价权下降=压缩开始。
竞争侧：从垄断→寡头过渡 — NVIDIA从90%+AI训练份额向70-80%压缩。TPU/Trainium/自研芯片在蚕食inference。但training仍是NVIDIA主场。

历史对标：

  • 不是2000年电信——AI终端需求(token生成)已经是real revenue，电信时期"带宽需求"尚未变现
  • 更像1995-1999年互联网——需求真实但规模预测偏高2-3x，基础设施先行但应用层尚在发展
  • 关键区别：NVIDIA毛利率75% vs 思科当年50-60%——当前链条的盈利能力远超历史
#11 什么在加速 vs 什么在减速
#12 周期转折的先导信号
  1. TSMC月度营收MoM下降连续2个月 → 需求放缓最早信号（领先NVIDIA财报1-2月）
  2. 任何大厂下调AI CapEx指引 → LOI基础松动
  3. NVIDIA毛利率跌破72% → 定价权被竞争侵蚀
  4. HBM现货价格下跌 → 存储供需反转
  5. ASML订单取消/延期 → 扩产预期修正（最滞后但最终确认性的信号）
PHASE V — BUILD

行业级交叉验证网络

#13 从"单人追踪"到"网络验证"

Jensen元思考的核心发现：单一信号源有偏见，多源交叉验证才可信。

行业级推广：20个代理人覆盖8个节点（含需求侧买方）。每个节点的龙头观点可以被其他节点的数据独立验证。

类比：20个基站的三角定位（含买卖双侧）比1个基站精确得多。
#14 最有价值的交叉验证对
#15 当所有人都同意时——最危险的信号

供给侧16人中没有一个公开bearish AI CapEx，需求侧4人(Altman/Nadella/Pichai/Hassabis)新增确认CapEx加速。所有人都在各自节点确认需求。

这本身就是一个风险信号：

  • 2000年电信泡沫：所有CEO都说"带宽需求无限" → 实际需求只兑现30%
  • 2007年房地产：所有银行家都说"房价只涨不跌" → 系统性风险被忽视
  • 当前AI CapEx：所有半导体CEO都说"需求insatiable" → 谁在做反面赌注？

答案：几乎没有人。这是一个拥挤的多头。不意味着一定会崩——但意味着一旦出现裂缝（大厂CapEx下调/LOI取消），下跌速度会超预期。

BULL 链1：AI CapEx持续 → 全链受益
需求验证 → NVDA(芯片) → TSM(代工) → ASML(光刻) → AMAT(设备) → SK hynix(HBM) → AVGO(网络)

PHASE VII — VERIFY

风险地图

#21 五个Kill Switches
#22 最容易被忽视的风险：AI应用ROI

原始16个代理人全部在谈供给侧（芯片/设备/封装）。新增4个需求侧代理人(Altman/Nadella/Pichai/Hassabis)提供了部分买方视角，但AI应用的ROI够不够支撑$700B+/年CapEx仍是核心未解问题。深度不对称：供给侧人均3+份专项素材 vs 需求侧刚建立。

如果AI应用层的变现速度慢于基建投入速度：

  • 2026-2027：大厂继续投（沉没成本+竞争恐惧）
  • 2028-2029：ROI审计→CapEx削减→需求悬崖
  • 这就是2000年电信的翻版——基建投完发现"用户还没来"

概率：完全崩塌~15-20%。规模修正(实际需求=预测的50-70%)~40-50%。

PHASE VIII — VERIFY

地缘政治

#23 Morris Chang + Chris Miller = 最权威的地缘分析

两人的共识：

"全球化已死" — Chang (TSMC创始人, 9/10可信) + Miller (Chip War作者, 7/10可信)。这不是修辞——CHIPS Act、出口管制、台海紧张是结构性变化。
"硅盾是脆弱的" — Chang: "依赖敌人的理性不是可靠的盾牌"。台湾的半导体垄断地位既是保护也是靶心。
"中国追赶在加速，但方向不同" — Jensen: 95%→50%份额下降。Chang: 先进制程差距5-6年在扩大。但成熟制程中国已经自给。

投资含义：市场给TSMC的台湾风险溢价(3-5%)太低。Chang暗示应该是8-10%。这意味着TSMC长期估值有下行风险。

#24 CHIPS Act的真实影响

20个代理人中涉及CHIPS Act的观点汇总：

  • Wei：Arizona成本+30-50%，但"战略价值而非经济价值"
  • Chang：从"futile"到"my dream"——接受了国安>经济
  • Lip-Bu Tan：Intel CHIPS Act拿了$78.6B承诺，但代工仍然不work
  • Dickerson：设备商受益——美国fab建设=更多AMAT订单

结论：CHIPS Act会建出产能，但成本高、效率低。真正受益的是设备商(AMAT/KLAC/LRCX)——他们不关心fab在哪里，只关心有没有人在建。

PHASE IX — SYNTHESIS

投资方法论

#25 七条核心原则
原则1：追踪链条，不追踪个股 — AI+半导体是7节点供应链。投资逻辑是"权力在哪个节点"，不是"哪个CEO说了好话"。
原则2：信Tier 1的方向，折扣Tier 4的时间线 — Wei/Chang说的直接用。Jensen打折30%。Musk只信方向(×2-3x时间线)。
原则3：共识=高置信度行动，分歧=投资机会 — 20人确认需求真实（含买方验证）→做多链条。Jensen说CUDA不可替代但5源矛盾→这是alpha信号。
原则4：看行动不看言论 — Jensen买Groq($20B) > 100次说"自研不合理"。Musk继续买GPU > 说"自建可替代"。收购/采购方向比财报电话诚实。
原则5：找"不管谁赢都赚"的位置 — TSMC/ASML/AVGO/SK hynix在bull和bear case都受益。优先于NVDA(单面)。
原则6：用时间差套利 — TSMC月度营收→NVIDIA季报→ASML年度订单。信号传导有6-9个月时间差。
原则7：所有人同意时加倍警惕 — 供给侧16人无一bearish，需求侧4人新增确认 = 仍偏拥挤多头。不意味着错，但意味着一旦裂缝出现，下跌速度超预期。
#26 持续监控体系

#27-#28 最终问题：我们最可能错在哪里？

最可能错的方向1：低估需求持续性
如果AI应用层变现速度超预期（GPT-5/Claude 5真的替代了大量白领工作），$1T不是高估而是低估。概率~25%。

最可能错的方向2：高估需求规模
如果AI应用ROI不达预期，$700B CapEx可能只兑现50-70%。整条链估值修正20-30%。概率~40%。

最可能错的方向3：忽视地缘黑天鹅
台海是所有模型都不愿定价的尾部风险。如果发生，不是修正而是重构。概率~5-10%/年。

最诚实的答案：方向大概率对（AI基建是真实的），规模可能偏高（×0.5-0.7），时间线可能偏早（×1.5-2）。持有链条核心（TSM/ASML/AVGO），用TSMC月度数据做动态校准。

PHASE X — BLIND SPOT

Missing Voice — 中国半导体认知盲区 The China Cognition Gap

20个西方代理人(现已扩展至30个含中国)的共识可能系统性低估中国半导体进展。这不是因为西方代理人不诚实，而是因为他们的利益结构决定了视角：卖家说"管制有效"比说"中国已经有替代了"更符合其商业利益。
A. 如果有中国代理人，哪些判断会改变？
B. 数据不对称风险 — "我们在最需要精确判断的维度上信息最少"

三层不透明 (3-Layer Opacity)

Company层 — SMIC良率、华为Ascend实际性能等核心数据不公开。外部只能通过拆机报告和间接推断。
Policy层 — 大基金实际拨付节奏、补贴条件、产能目标等关键政策信息高度不透明。
Narrative层 — 双方都在spin：中国媒体夸大突破，西方媒体夸大管制效果。真实进度在两者之间。

⚠ 操作建议

所有涉及中国半导体的概率判断，使用 ±15% 置信区间。例如"SMIC 7nm良率50%"应理解为"35%-65%区间内"。

C. 认知盲区的结构性原因
1. 语言障碍 — 中文财报、采访、政策文件的可及性。大量一手信息存在于微信公众号、财新等西方分析师不常访问的渠道。
2. 利益冲突 — 西方分析师的客户是西方投资者。"管制有效→NVIDIA安全→继续买"比"中国有替代→NVIDIA TAM缩小"更受客户欢迎。
3. 信息管控 — 中国政府对半导体信息的分类管理。SMIC不再公开先进制程细节，华为不公布Ascend具体出货量。
4. 时间延迟 — 中国半导体突破通常在公开报道前6-12个月就已发生。等西方媒体报道时，已是旧闻。

🔎 关键问题

如果中国半导体自给率在2027年达到50%而不是当前估计的30-35%，我们的整个投资框架需要如何调整？

这意味着：NVIDIA的中国收入永久归零（已发生），HBM需求被国产DRAM部分替代，TSMC先进制程溢价下降（竞争者出现），设备商面临出口管制升级风险。唯一不变的：ASML的EUV垄断（但DUV多重曝光可能侵蚀部分需求）。
