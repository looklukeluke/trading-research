4. 一手资料Primary Sources

Bitcoin Whitepaper (2008) — 创世白皮书

The Cryptography Mailing List, October 31, 2008

Satoshi Nakamoto宣布比特币诞生的9页论文，引用了B-money (Wei Dai) 和 Hashcash (Adam Back) 的工作，提出了一个点对点的电子现金系统。这是理解Satoshi设计意图的最权威起点。

URL: bitcoin.org/bitcoin.pdf ↗

Satoshi Nakamoto Institute — 档案库

最完整的Satoshi公开言论档案库，由Pierre Rachet创建并维护。包含：

  • 所有Bitcointalk论坛帖子 (2009-2010)
  • 所有Cryptography邮件列表通信
  • P2P Foundation帖子
  • SourceForge代码提交记录

URL: satoshi.nakamotoinstitute.org ↗ 这是研究Satoshi言行不可替代的权威来源。

Genesis Block — 创世区块

2009年1月3日挖出的第一个区块，包含《泰晤士报》的标题："Chancellor on brink of second bailout for banks"（财政大臣正处于第二轮银行纾困的边缘）。

这是比特币与现实世界连接的第一次时间戳，不可篡改的历史证据。

Block Hash: 000000000019d6689c085ae165831e934ff763ae46a2a6c172b3f1b60a8ce26f

Bitcointalk.org — 早期论坛存档

Satoshi亲自创建并发帖的论坛，2009-2010年活动记录。Satoshi在该论坛的ID是 "satoshi"，于2009年11月22日注册。

URL: bitcointalk.org ↗

SourceForge / GitHub 早期代码

原始代码提交记录，Satoshi作为维护者的技术轨迹。可以看到从0.1.0版本到0.3.24 (2011年4月) 的所有提交。

URL: GitHub ↗

5. 关键关系网Key Relationships

Hal Finney — 最密切的合作者 / Closest Collaborator

关系强度：⭐⭐⭐⭐⭐

密码学家、开发商，2009年1月12日收到Satoshi发送的第一笔比特币交易。2014年因渐冻症去世。

Hal Finney曾被长期怀疑是Satoshi本人，但他本人多次否认，死后家人也证实他并非中本聪。

关键贡献：参与比特币代码审查、帮助Satoshi处理早期技术问题、构建了RPOW系统。

Gavin Andresen — 指定的继任者 / Designated Successor

关系强度：⭐⭐⭐⭐⭐

Satoshi于2010年将代码库维护权限移交给Gavin。2010-2011年间是唯一与Satoshi有直接邮件往来的人。

Gavin后来成为比特币社区的重要声音，但在2016年因支持比特币XT分叉而与Core团队分道扬镳。

Nick Szabo — 思想先驱 / Intellectual Precursor

关系强度：⭐⭐⭐⭐☆

"智能合约"概念提出者，Bit Gold设计者（1998年）。Satoshi在白皮书中引用了他的工作。

Nick Szabo是Satoshi身份的主要猜测对象之一，但缺乏直接证据。

Wei Dai — 理论奠基 / Theoretical Foundation

关系强度：⭐⭐⭐⭐☆

B-money提出者（1998年），Satoshi在白皮书参考文献中提及。两人有过邮件往来。

Adam Back — 技术桥梁 / Technical Bridge

关系强度：⭐⭐⭐⭐☆

Hashcash发明者（1997年），比特币工作量证明的基础。2015年创立Blockstream。

Craig Wright — 争议人物 / Controversial Figure

关系强度：⭐⭐☆☆☆

澳大利亚计算机学家，自2016年起自称Satoshi。引发众多法律诉讼，但主流学界和社区普遍不接受其声称。

Wright的案例说明了一个关键问题：在密码学世界，证明身份需要的是密钥而非法律文件。

Dorian Nakamoto — 错误指认 / Mistaken Identity

关系强度：⭐⭐☆☆☆

2014年Newsweek的错误报道将其指认为Satoshi，实际是完全无关的日裔美国人工程师。

6. 关键叙事弧Key Narrative Arcs

叙事弧 #1: 从密码朋克到比特币

理解Satoshi必须理解其思想谱系：David Chaum的盲签名 → Wei Dai的B-money → Nick Szabo的Bit Gold → Adam Back的Hashcash → Bitcoin。这一脉络展示了Satoshi如何在前人基础上实现了关键突破。

关键洞察：Satoshi不是凭空创造，而是"站在巨人肩膀上"的系统性整合。

叙事弧 #2: 消失的艺术家

Satoshi于2010年12月12日在Bitcointalk发布最后一个技术帖子后逐渐淡出。2011年4月最后一次代码提交后完全消失。这种"优雅退出"是历史上最完美的公关——留下一个永远无法被攻击的匿名者。

时间线：2010年12月(最后公开论坛发言) → 2011年4月(最后代码提交) → 2014年( Dorian新闻周刊事件) → 至今(仍未现身)

叙事弧 #3: 匿名即战略资产

Satoshi选择匿名的动机：防止政府打压、避免成为"单点故障"、让比特币去中心化属性不受创始人个人影响。这与中世纪隐士数学家（如发现群论的伽罗瓦）有异曲同工之妙。

战略悖论：越是匿名，越显得重要；越是不露面，越多人追寻。这种"稀缺性注意力"成为比特币神话的核心。

叙事弧 #4: 100万BTC的沉默

根据链上分析，Satoshi在2009年挖出约100万BTC，至今未动。这笔财富在2021年峰值时价值超过600亿美元，但从未转移。

这成为证明Satoshi身份的唯一"验钞机"：如果有人能移动这些币，那几乎可以肯定是中本聪本人。
