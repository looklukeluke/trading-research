Node 5: 定制芯片/ASIC — 28轮元思考过程

从"ASIC vs GPU经济学"到"谁赢Broadcom都赢"  |  2026-03-20
研究笔记 — 思考过程比结论更有价值

28轮元思考 9个Phase 6个claims交叉验证 AVGO/MRVL/TSLA/AAPL

🧠 28轮元思考的核心结论

  1. 自研芯片的赢家不是自研者本身，是Broadcom（设计服务~80%份额）和TSMC（代工）。Google、Apple、Meta、Tesla全在自研，但全部仍需Broadcom的设计能力和TSMC的制造能力
  2. ASIC vs GPU不是替代关系，是互补关系。ASIC增速44.6%远超GPU的16.1%，但绝对规模仍远小于GPU。自研补充通用GPU，不替代它。Jensen说"自研不合理"是5%可信——但他说"需求继续爆发"是90%可信
  3. Hock Tan是半导体行业最被低估的CEO信号源。他的ASIC收入数据是AI自研趋势最直接的温度计。AVGO AI收入$84B/Q +106%——这不是叙事，是硬数字
  4. 最有价值的投资方法论：当无法判断Musk/Srouji/Keller谁的路径胜出时，买"无论谁赢都受益的军火商"——AVGO/TSM/ASML
PHASE I — EXPLORE

任务定义：这个节点在研究什么？

#1 表面任务 vs 真正任务

表面任务：分析Custom Silicon/ASIC节点——Google TPU、Apple自研、Tesla TeraFab等。

真正任务：为趋势交易者回答一个核心问题——Hyperscaler自研浪潮是NVIDIA的终结还是AVGO的春天？自研趋势对GPU供应链的侵蚀速度和受益者分布是什么？

交易者需要判断：该买NVDA（赌自研失败）还是AVGO（赌自研成功但不影响设计服务）还是两边对冲？

#2 ASIC vs GPU经济学——核心框架

GPU优势：通用性高、软件生态(CUDA 6M+开发者)、迭代速度(年度cadence)、即插即用。

ASIC优势：特定工作负载TCO低3-5x、功耗效率高、推理场景高度优化。

关键判断：当推理需求100x爆发时，ASIC的经济学优势呈指数放大——推理工作负载高度可预测，正是ASIC最擅长优化的场景。但训练仍然是GPU的天下。

#3 三个代理人为什么选Musk/Srouji/Keller

Musk（xAI/Tesla）：最激进的自研路线——从芯片设计到TeraFab自建工厂。他是"自研到底"的极端测试。

Srouji（Apple Silicon）：最成功的自研案例——16年零重大延迟。他是"自研最优执行"的标杆。但即使Apple也与Broadcom合作Baltra。

Keller（Tenstorrent/RISC-V）：最颠覆的IP授权模式——开源+模块化。他是10年时间尺度的结构变化。

选这三人不是因为他们是"ASIC公司CEO"，而是因为他们代表自研光谱的三个极端。

#5 Broadcom的XPU商业模式解剖

Broadcom的ASIC设计服务商业模式是整个节点的核心。Hock Tan创建了一个"军火商"模型：

  • 设计费：每个ASIC设计项目$100-300M+，周期18-36个月
  • 持续版税：量产后每颗芯片抽成，规模效应随客户出货量增长
  • 锁定效应：客户一旦进入设计流程，切换成本极高（2-3年sunk cost）
  • 产能瓶颈：Broadcom设计团队产能本身成为供给约束——不是不想接更多客户，是没有足够的工程师

关键洞察：Broadcom的瓶颈不是需求而是供给。这意味着MRVL（第二tier）的机会来自Broadcom的"溢出"。

#6 Google TPU十年史——最被忽视的信号

Google TPU从2015年立项至今已10年。它是自研ASIC的最长跑道验证：

  • 成功：TPU v6推理成本领先NVIDIA 40-60%，Anthropic Claude在TPU上训练
  • 局限：仍然仅内部使用，未对外商用。Google Cloud TPU份额远低于NVIDIA GPU
  • 含义：即使10年+数十亿投入，自研ASIC仍有"规模天花板"——只能服务自己的工作负载

这对Jensen是好消息（TPU不会替代通用GPU市场），但对AVGO也是好消息（更多公司会学Google自研）。

PHASE III — DIAGNOSE

矛盾与张力：自研浪潮的四个裂缝

#7 四个核心张力
张力1：每家都自研 vs 每家仍买GPU — Google/Apple/Meta/Tesla/Amazon全在自研芯片。但同时每家都继续大量购买NVIDIA GPU。"自研替代GPU"的叙事忽略了关键事实：自研ASIC瞄准的是推理/特定工作负载，通用训练仍需GPU。两者是互补，不是替代。
张力2：Musk的"全栈"vs Srouji的"最优分工" — Musk要从芯片到工厂全自研（TeraFab+Samsung）。Srouji坚持自研设计+TSMC代工+Broadcom合作。16年零延迟 vs "大力出奇迹"——哪个是正确的自研深度？对投资者含义巨大：Musk路径成功=TSMC受损，Srouji路径成功=TSMC+AVGO受益。
张力3：Broadcom的"不可替代" vs Keller的"开源颠覆" — Broadcom设计服务~80%份额看似垄断。但Keller的Tenstorrent提出开源IP授权模式，让更多公司不需要Broadcom就能设计芯片。$150M vs $84B/Q——规模差距3个数量级，但方向可能正确。这是10年时间尺度的威胁。
张力4：Jensen否认 vs Jensen行动 — Jensen公开说"自研芯片不合理"——然后花$20B收购Groq（一家非GPU公司做推理ASIC）。Jensen的言行不一是Node 5最强的反向验证：他自己用行动承认了ASIC在推理场景的价值。
#8 信息空白——我们不知道什么
  1. Hyperscaler自研芯片的真实TCO比较——每家都声称"比GPU便宜X倍"，但没有独立第三方验证
  2. Broadcom设计管线深度——Hock Tan说"8个客户在设计中"但不公布具体进度
  3. Tesla TeraFab的真实良品率——Musk从未公开任何晶圆制造良品率数据
  4. Apple Baltra的架构细节——Apple对其首款服务器芯片的技术规格严格保密
  5. 自研 vs 外购的真实CapEx分配比例——每家Hyperscaler都在同时投入两条路线，比例不公开

结论：自研领域的公开信息系统性不足。最可靠的信号来自Broadcom/TSMC的收入数据——这是硬数字，不是叙事。

PHASE IV — DIAGNOSE

时间与反面：如果自研浪潮是泡沫？

#9 什么在加速/减速？

加速中(看多ASIC)：推理需求100x爆发 | Hyperscaler CapEx $150B+/年 | ASIC增速44.6% | Broadcom管线8+客户

减速中(看空ASIC)：自研项目2-3年才出成果 | TeraFab验证周期漫长 | Tenstorrent规模微小 | GPU年度cadence迭代更快

#10 最强反面论证——自研浪潮是"me-too"泡沫
每家Hyperscaler自研芯片不是因为经济学最优，而是因为"别人都在做，我不能落后"。
  1. "自研"的隐性成本被低估：除了芯片本身，还需要编译器、运行时、工具链、人才——Google TPU花了10年才成熟
  2. NVIDIA年度cadence让自研永远追不上：自研ASIC从设计到量产2-3年，但NVIDIA每年推新架构。自研者在追一个移动靶
  3. 大部分自研项目会缩减：2-3年后，当ROI不如预期，部分Hyperscaler可能砍掉或缩减自研团队，回归买GPU
  4. Intel自研经验的警告：Intel曾尝试自研一切，最终证明"什么都做=什么都做不好"

概率：15-20%。自研趋势的方向性是确定的，但规模预测可能偏高2x。即使部分项目失败，AVGO仍然受益（已收到的设计费不退）。

#11 TeraFab——最极端场景分析

如果TeraFab成功（25%概率）：

  • Tesla证明非半导体公司可以自建晶圆厂
  • 更多公司尝试垂直整合 → TSMC的"不可替代性"被质疑
  • 但设计服务仍需要AVGO——建工厂和设计芯片是不同技能

如果TeraFab失败（75%概率）：

  • 验证"专业分工"才是正确路径 → Srouji模式(设计自研+TSMC代工)成为标准
  • AVGO和TSMC的"不可替代性"进一步巩固

结论：TeraFab成败对交易策略的影响是不对称的——失败(75%)强化现有论文，成功(25%)需要重新评估整条供应链。

#13 代理人选择的元思考

为什么不选Hock Tan作为代理人？

Hock Tan是AVGO CEO，是Node 5最直接的信号源。但他的视角有系统性偏向——永远在推销ASIC设计服务的增长。选择Musk/Srouji/Keller是为了从客户/竞争者/颠覆者三个角度交叉验证ASIC趋势。

Jensen作为"竞争对手视角"：Jensen否认自研的价值，但Groq收购暴露了他的真实判断。他的言行矛盾本身就是Node 5最强的验证信号。

PHASE VI — VERIFY

成品审视：诚实的自我诊断

#14 做到了什么/没做到什么

做到了：三代理人交叉验证有效、"军火商"论文清晰、AVGO/MRVL投资逻辑明确。Phase 0诚实标注了"事后重构"。信号传导链三条(Bull/Bear/TeraFab)覆盖了主要场景。

没做到：ASIC vs GPU的TCO比较缺乏硬数据（每家自研者都声称降本但没有独立验证）。Marvell的分析深度不如Broadcom（MRVL代理人位置被Keller占了）。Tesla TeraFab的技术可行性分析偏粗——缺乏晶圆制造良品率的专业评估。

#15 Broadcom $84B/Q——数字的可靠性检查

AVGO AI收入$84B/Q被重复引用。但需要注意：

  • 这是"AI半导体收入"，不全是ASIC设计服务——包含networking芯片(Tomahawk/Jericho)和其他
  • Hock Tan对"AI收入"的定义比较宽泛——有些收入在其他公司会归为"数据中心"
  • 设计服务本身的收入更低，但确切比例AVGO不公布

修正：报告中保留$84B/Q AI收入但标注这包含网络芯片。纯ASIC设计服务收入可能是$30-40B/Q量级。

#16-#17 Padding识别 + 叙事偏见检查

Padding：Musk传记细节（与芯片无关的SpaceX/Tesla故事）、Srouji的Apple历史（不影响投资判断）、RISC-V技术架构细节（过于技术）。

叙事偏见：报告整体偏向"AVGO是赢家"的结论。需要更强调：(1) AVGO估值已反映部分预期；(2) 如果AI CapEx整体崩塌，ASIC受损不比GPU少；(3) Broadcom的"军火商"模式在客户自研能力成熟后会萎缩。

PHASE VII — BUILD

季度验证日历

#18-#19 预测追踪器

交易者不关心"自研芯片技术进展"——关心的是"下个季度AVGO的AI收入能不能维持100%+增长"。

#20 四个交易机会
机会1：AVGO财报方向性交易。Broadcom每季度AI收入增速是ASIC趋势最硬的温度计。链条：AVGO→TSM→MRVL→SNPS/CDNS（EDA工具）
机会2：Apple Baltra里程碑（Q3-Q4 2026）。如果按时推进=Broadcom合作模式最强验证。Apple的执行力是Srouji16年的信用积累。
机会3：MRVL作为AVGO的"溢出"受益者。如果Broadcom产能满载，Tier-2客户流向Marvell。需要第二个Tier-1客户win作为催化剂。
机会4：AVGO vs NVDA相对价值交易。如果自研加速→AVGO相对NVDA跑赢；如果自研放缓→NVDA相对跑赢。配对交易隔离方向风险。
#21 监控方法论

每月：TSMC月度营收(10日) | Hyperscaler CapEx新闻 | 芯片人才流动(LinkedIn信号)

每季度：AVGO财报(AI收入增速) | MRVL财报(AI收入增速) | TSLA earnings(TeraFab提及)

事件驱动：Apple产品发布(Baltra) | Tenstorrent融资/客户公告 | Jensen关于自研的新言论

PHASE VIII — BUILD

信号传导与标的地图

#22 三条传导链
BULL 链1：自研加速 → 设计服务链
更多公司自研 → AVGO(设计) → TSM(代工) → MRVL(溢出) → ARM/RISC-V(IP) → SNPS/CDNS(EDA)
信号最先在AVGO AI收入出现，最后传到TSMC先进制程利用率和ASML设备订单
BEAR 链2：自研失败 → 回归GPU链
自研项目缩减 → 回购NVIDIA GPU → NVDA受益 → AVGO设计收入下降 → MRVL受冲击更大
信号最先在Hyperscaler CapEx分配变化出现。关键：AVGO在两条链都能存活（设计服务+网络芯片）
WATCH 链3：TeraFab成功 → 产业链重构
Tesla证明自建工厂可行 → 更多公司尝试垂直整合 → TSMC"不可替代性"被质疑
概率最低(25%)但影响最大。信号最先在Tesla CapEx和TeraFab良品率数据出现
#23 时间差套利

当AVGO AI收入持续100%+增速但MRVL还没有第二个Tier-1 win——这可能是MRVL的预期差窗口。

#24 "不管谁赢都赚"标的

对比：NVDA只有自研放缓才大赢。TSLA只有TeraFab成功才大赢。MRVL需要突破Tier-1客户。

#25 历史案例
Google TPU (2015-2026)：10年最成熟的自研案例。推理成本领先40-60%，Anthropic在TPU训练。但仍未对外商用。自研可以成功，但有规模天花板。
Apple Silicon (2010-2026)：最完美的自研执行。M1→M5零重大延迟，Baltra进军服务器。但即使Apple也需要Broadcom合作和TSMC代工。
Jensen买Groq ($20B, 2025.12)：最强反向信号。嘴上否认ASIC价值 → 手上$20B买推理ASIC公司。当对手用行动承认你的价值时，这是最高置信度的验证。
PHASE IX — VERIFY

交叉验证网络 — 最终方法论

#26 三个代理人如何交叉验证？

共识区域(高置信度)：三人都在用行动证明自研可行——Musk建TeraFab，Srouji做Baltra，Keller卖IP。自研趋势不可逆。

分歧区域(需要跟踪)：垂直整合深度完全不同——Musk要自建工厂，Srouji坚持专业分工，Keller想颠覆设计服务模式。这个分歧直接决定AVGO(Broadcom)的长期价值。

类比：三个建筑师都同意"需要盖房子"(自研趋势)，但对"谁来建"(自研深度)意见完全不同。投资者应该买"建材供应商"(AVGO/TSM)而不是赌哪个建筑师的方案胜出。
#27 6个Claims的交叉验证结果

最终方法论 — 4条规则

规则1："军火商"优先于"战士"
当无法判断哪条自研路径胜出时，买"无论谁赢都受益的供应商"。AVGO/TSM/ASML是"军火商"，NVDA/TSLA是"赌方向的战士"。

规则2：用Broadcom收入验证趋势，不用CEO叙事
Hock Tan说"$60-90B AI收入"——这需要验证。但每季度的实际AI收入是硬数字。AVGO财报日是ASIC趋势最准的温度计。

规则3：Jensen的行动比言论更有价值
他否认自研 → 买Groq。他否认CUDA可替代 → 加速NIM/Dynamo锁定。当CEO用行动否定自己的公开言论时，信行动。

规则4：监控"溢出"信号
当AVGO产能满载信号出现（设计周期延长、客户排队），MRVL是第一个溢出受益者。但需要Tier-1客户win作为催化剂确认。
