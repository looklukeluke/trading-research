光刻设备 Node 2 — 28轮元思考过程

ASML EUV垄断的可投资性评估  |  2026-03-20
研究笔记 — 思考过程比结论更有价值

28轮元思考 9个Phase 7个claims交叉验证 3条传导链

🧠 28轮元思考的核心结论

  1. ASML不是一个股票信号，是整个半导体CapEx周期的温度计。追踪ASML的订单积压(€38.8B)比追踪任何分析师预测都准——因为那是签了合同的真金白银
  2. Fouquet的保守主义是系统性的，可以套利。2025年7月说"cannot confirm 2026 growth"导致$30B蒸发→5个月后交出创纪录订单。他的悲观=逆向买入信号
  3. High-NA是ASML最大的叙事风险。TSMC选择跳过High-NA用多重曝光标准EUV做2nm/1.4nm——全球最先进foundry不需要你最贵的新产品，这个信号必须认真对待
  4. "不管谁赢都赚"的纯度最高：NVIDIA赢、AMD赢、Google TPU赢、甚至Intel翻身——都需要ASML的光刻机。唯一输的场景是AI CapEx整体崩塌
PHASE I — EXPLORE

任务定义：光刻设备节点的真正问题

#1 表面任务 vs 真正任务

表面任务：将光刻设备数据页升级为认知代理人报告。

真正任务：评估ASML EUV垄断作为投资标的的可靠性和时机——不是"ASML好不好"(所有人都知道好)，而是"现在买贵不贵、什么信号意味着拐点、High-NA到底值不值"。

交易者需要的不是ASML传记，而是：订单积压的硬度、Fouquet保守主义的套利空间、以及出口管制对收入的真实影响。

#2 三个代理人的选择逻辑

Fouquet (ASML CEO) = GIANT：100% EUV垄断者，看到全球每一个先进fab的设备采购计划。信息优势9/10。

Dickerson (AMAT CEO) = INVESTOR：最大的广谱设备商，从不同角度验证设备周期。"Super cycle"叙事的独立来源。

Chris Miller (Chip War作者) = CONTRARIAN：地缘政治学者，从chokepoint角度审视ASML垄断——垄断既是定价权也是地缘脆弱性。

三角定位：供给侧(Fouquet) + 需求侧(Dickerson) + 风险侧(Miller)。

#3 成功/失败标准

成功：报告第一眼回答"ASML的€38.8B积压订单有多硬"；有Fouquet保守主义的交易价值分析；有出口管制的收入影响量化。

失败：变成"EUV好厉害"的技术科普(没有投资洞察)；忽略High-NA争议(只讲好话)；没有与AMAT/LRCX的对比框架。

PHASE II — EXPLORE

素材解码：三个代理人讲了什么故事？

#4 核心叙事：垄断者的三重视角

Fouquet视角(供给)：€38.8B积压订单排到2027。EUV 100%垄断。中国收入从49%降至27%已被消化。High-NA开始贡献但采纳比预期慢。系统性保守——"cannot confirm 2026 growth"后5个月交出创纪录订单。

Dickerson视角(需求)：WFE市场$130B+(2026)进入"super cycle"。不止光刻——GAA晶体管、先进封装、HBM设备全部受益。AMAT R&D增速10.42%超过收入增速4%=提前投资。EPIC Center $5B——Samsung/SK Hynix/Micron同时签约。

Miller视角(风险)：ASML是全球供应链最关键的单点故障。Zeiss镜头零替代。出口管制从EUV扩展到DUV是最大下行风险。中国自研光刻(SMEE)在90nm——落后15+年。"firmly out of reach"。

跨视角交叉信号

#5 信号 vs 噪音筛选

高信号：订单积压金额、High-NA采纳率、出口管制范围、TSMC设备选择、Fouquet tone变化。

低信号(删减)：ASML历史回顾(科普)、EUV技术原理(教科书)、中国SMEE进度(太远)、Zeiss公司介绍(背景)。

PHASE III — DIAGNOSE

矛盾与张力：垄断者的裂缝

#6 四个核心张力
张力1：保守指引 vs 创纪录交付 — Fouquet 2025年7月说"cannot confirm growth"→5个月后€13.2B创纪录订单(近2x预期)。系统性under-promise——但市场每次都信。这个模式本身是交易机会。
张力2：EUV垄断 vs High-NA采纳困境 — ASML花了数十亿开发High-NA——但最大客户TSMC选择跳过。Intel(亏损中)和Samsung(追赶中)是仅有的买家。垄断者的新产品找不到最大客户。
张力3：地缘武器 vs 商业利益 — 出口管制让ASML变成地缘政治工具。中国收入从49%降到27%——ASML被迫放弃全球第二大市场。进一步DUV禁令可能再削10-15%。垄断的政治代价正在显现。
张力4：Dickerson的"Giga-cycle" vs 历史周期 — Dickerson从"tipping point"升级到"super cycle"再到"giga-cycle"。每个比前一个更牛。问题：哪些是订单数据支撑的，哪些是财报电话会上的营销话术？
#7 信息空白——我们不知道什么
  1. High-NA真实采纳曲线——TSMC跳过了，Intel/Samsung的采购量是否足以支撑投资回报？
  2. DUV出口管制扩展时间表——欧洲智库建议≠政策，但方向明确
  3. 中国替代进度的真实状态——公开信息说90nm(SMEE)，但军方项目不公开
  4. Zeiss产能上限——ASML的瓶颈不在自己而在Zeiss的光学镜头产能
  5. €38.8B积压的取消条款——合同里有多少flexibility？客户能不能推迟交付？

结论：ASML的垄断比Jensen的预测更"硬"(签合同vs说话)，但出口管制和High-NA采纳是两个不可忽视的结构性不确定性。

PHASE IV — DIAGNOSE

时间与反面：如果EUV垄断被侵蚀？

#8 什么在加速/减速？

加速中(看多)：AI CapEx驱动WFE $130B+ | 积压订单€38.8B | AMAT/LRCX同步创新高 | 每个先进节点需要更多EUV层

减速中(看空)：High-NA采纳率低于预期 | 中国收入永久性下降 | DUV禁令风险 | ASML从52周低已涨~140%

#9 最强反面论证
ASML不是"不可替代的垄断"，是"被地缘政治绑架的单点故障"。
  1. High-NA是ASML的"Concorde时刻"——技术上成功但商业上可能不够成功。TSMC跳过=最大客户投票"不需要"
  2. 出口管制让ASML永久失去中国市场——49%→27%→如果DUV也禁可能到15%。这不是周期波动，是结构性市场收缩
  3. Zeiss是真正的瓶颈——ASML的产能上限由一家德国私有公司的光学镜头产能决定。单点故障中的单点故障
  4. WFE super cycle ≠ ASML super cycle——先进封装(CoWoS)设备增长更快，AMAT/TEL的封装设备增速可能超过光刻
  5. 估值已反映乐观——从52周低点涨140%，Forward P/E ~30x。好消息已在价格里

概率：15-20%。EUV垄断在物理上无可撼动(5-10年)，但High-NA和出口管制确实压缩了增长天花板。

PHASE V — BUILD

报告结构设计

#10 结构决策

核心原则：报告不是"光刻科普"，是ASML/AMAT/LRCX作为设备周期投资标的的可靠性评估。

  1. Phase 0: 独立思考——光刻设备解决什么问题、供需状态、EUV垄断性质
  2. Phase 0.5: 策展搜索——Fouquet/Dickerson/Miller的一手材料
  3. Phase 1: 三个代理人——行动证据优先，认知时间线，可信度评分
  4. Phase 2: 综合——三角验证表、竞争格局、供需框架、风险矩阵、交易视角
  5. KPI + 图表 + 溯源

Phase 0偏差标记：(1)高估中国DUV替代速度 (2)未预见TSMC跳过High-NA。两个方向性错误。

#11 Phase 0偏差的投资意义

偏差1(中国DUV高估)：CSIS评估"firmly out of reach"比预期强得多。SMEE在90nm=落后ASML 15+年。中国半导体自给率仅~30%远低于70%目标。投资含义：短期内(3-5年)中国不构成技术替代威胁，ASML垄断更硬。

偏差2(High-NA低估)：没有预见TSMC选择多重曝光标准EUV跳过High-NA。投资含义：High-NA的可寻址市场缩小到Intel(挣扎中)+Samsung(追赶中)+研发机构(imec)。这实质性降低了ASML下一代收入的上行空间。

两个偏差方向相反：一个让垄断更硬(bull)，一个让增长更慢(bear)，净效应大致中性——但风险构成与Phase 0预测不同。

PHASE VI — VERIFY

成品审视：诚实的自我诊断

#12 做到了什么/没做到什么

做到了：三个代理人的行动证据+认知时间线完整；三角验证表清晰展示共识(EUV垄断)和分歧(High-NA/出口管制)；竞争格局表覆盖ASML/AMAT/LRCX/TEL。

没做到：Zeiss依赖性分析不够深(缺少产能数据)；LRCX的独立视角不够(只是跟随ASML/AMAT的验证)；出口管制的政策分析过于定性缺少量化模型。

#13-#15 Fouquet保守偏见的交易价值

Fouquet的保守主义不是性格特征，是系统性的公司治理策略。

模式识别：2025.07"cannot confirm growth"(股价暴跌$30B) → 2025.12 创纪录€13.2B订单。公开悲观 + 订单积压创新高 = 模式性under-promise-over-deliver。
交易规则：当Fouquet转悲观 + 基本面(积压/利用率)无恶化 → 逆向买入窗口。关键区分：Fouquet"真悲观"(积压下降) vs "习惯性悲观"(积压稳定/上升)。
可信度校准：当季度 9/10(他有数据) | 下季度 8/10(保守但准确) | 2027+ 6/10(技术转换真正不确定)
#16 Dickerson"Super Cycle"的拆解

Dickerson用三个递进术语：tipping point → super cycle → giga-cycle。

有数据支撑的：Q1 FY2026 $7.01B创纪录 | R&D增10.42%超收入增4% | EPIC Center $5B实体投资 | 三家存储公司(Samsung/SK Hynix/Micron)同时签约——super cycle有基础。

存疑的："Giga-cycle"首次出现在Q1财报电话会=最大化股价影响的时机。裁员4%被包装为"geographic rebalancing"但也服务利润率优化。

三角验证结论：super cycle是真的(Fouquet订单数据+多家Hyperscaler CapEx确认)。giga-cycle是aspirational framing。折扣Dickerson叙事~20%，用Fouquet积压作为ground truth锚。

PHASE VII — BUILD

季度验证日历

#18-#19 预测追踪器

光刻设备的验证周期比GPU长——设备订单到交付有12-18个月滞后。交易者需要的领先指标不是ASML自己的财报，而是下游客户的CapEx指引。

#20 三个投资机会
机会1：Fouquet保守主义套利。模式：公开悲观tone+积压不减=逆向买入。下一次机会：Fouquet在某次财报上"manage expectations"
机会2：设备周期扩散。ASML→AMAT→LRCX→TEL——信号在ASML订单最先出现，在TEL最后反映。时间差2-3季度
机会3：出口管制恐慌后的超跌。DUV禁令消息→ASML下跌10-15%→但基本面(非中国积压)未变=买入窗口
#21 监控方法论

每月：TSMC月度营收(10日)——光刻设备需求的最直接前瞻指标

每季度：ASML积压变化 | AMAT订单趋势 | TSMC/Intel/Samsung CapEx指引

事件驱动：DUV出口管制政策变化 | High-NA大额订单公告 | TSMC新节点设备选择

PHASE VIII — BUILD

信号传导与标的地图

#22 三条传导链
BULL 链1：AI CapEx上行 → 设备链
Hyperscaler CapEx → TSMC/Intel/Samsung扩产 → ASML(EUV) → AMAT(沉积/刻蚀) → LRCX(刻蚀/沉积) → TEL(涂胶/显影)
信号最先在TSMC月度营收出现，ASML订单滞后1-2季度，TEL滞后2-3季度
BEAR 链2：出口管制升级 → 收入收缩链
DUV禁令扩大 → ASML中国收入再降10-15% → ASML短期股价冲击 → 但Zeiss/AMAT中国收入也受影响
信号最先在政策公告出现。ASML反应最快(直接暴露)，AMAT次之
WATCH 链3：先进封装崛起 → 设备结构变化
CoWoS需求爆发 → 封装设备(AMAT/TEL/Besi)增速超光刻 → WFE份额从光刻向封装倾斜
这不是看空ASML，而是AMAT/Besi可能跑赢ASML
#23 时间差套利

最可行的时间差：TSMC月度营收↑ → ASML下季度积压↑ → 在TSMC数据公布后、ASML财报前建仓。

#24 "不管谁赢都赚"标的

ASML是六个半导体节点中"不管谁赢都赚"纯度最高的标的。唯一输的场景是AI CapEx整体崩塌——但那种情况下所有半导体都输。

#25 历史案例
2025.07 Fouquet"cannot confirm"事件：市值蒸发$30B。5个月后Q4创纪录订单。追踪者如果识别了保守主义模式→跌幅即买点。
2024 中国收入49%：市场恐慌"如果中国被切断"。实际执行：中国从49%降至27%，但非中国订单€38.8B填满。ASML的垄断地位意味着需求不是消失而是转移。
AMAT EPIC Center $5B：不是口号而是实体投资。Samsung/SK Hynix/Micron三家作为founding partner=独立验证设备需求是真的。
PHASE IX — VERIFY

交叉验证网络 — 最终方法论

#26 设备节点的独特验证方法

光刻设备与GPU不同——设备有12-18个月的订单-交付滞后。这意味着：

ASML的积压订单是整个半导体行业最"硬"的前瞻指标。€38.8B不是预测，不是LOI，是签了字的合同。比任何CEO的口头预测都可靠。

但"硬"不意味着"不变"——需要监控取消/推迟率。历史上CapEx下行周期中积压取消率可达10-15%。

#27 7个Claims的交叉验证结果

最终方法论 — 光刻设备投资4条规则

规则1：ASML积压订单是Ground Truth
€38.8B签约合同 > 所有分析师预测。积压增长=加仓信号。积压下降=减仓信号。取消率超过5%=警报。

规则2：Fouquet悲观 + 积压稳定 = 逆向买入
系统性保守 + 市场过度反应 = 重复出现的交易机会。关键：区分"习惯性悲观"(积压稳定)和"真正悲观"(积压下降)。

规则3：High-NA是期权不是确定性
TSMC跳过了。Intel/Samsung可能买。但不要把High-NA当作确定的增长引擎。把它当做上行期权——如果兑现是bonus，不兑现不影响核心论据。

规则4：出口管制恐慌是买入机会(有条件)
条件：非中国积压保持稳定。如果DUV禁令导致ASML暴跌10%+，但€30B+非中国积压不变——这是市场给的礼物。
