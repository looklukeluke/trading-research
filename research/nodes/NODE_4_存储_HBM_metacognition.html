存储/HBM Node 4 — 28轮元思考过程

HBM超级周期 vs 经典存储周期的终极博弈  |  2026-03-20
研究笔记 — 思考过程比结论更有价值

28轮元思考 9个Phase 7个claims交叉验证 3条传导链

🧠 28轮元思考的核心结论

  1. HBM是所有半导体节点中"供需确定性最高"的。2026年全部产能已预售完毕。三家CEO全部大幅加CapEx=用真金白银投票。这不是猜测，是合同
  2. 真正的问题不是"供不应求有没有"(有)，而是"能持续多久"。Kwak说"到2030年"，Mehrotra说"超过2027年"。2027-2028三家同时扩产上线是最大的周期拐点风险
  3. Samsung HBM4获NVIDIA最高评分是格局变化信号。HBM3E时代SK Hynix绝对主导(50%+)。HBM4时代可能是三强争霸。这改变了定价权分布
  4. "这次不同"的叙事要严肃对待但也要警惕。5年SCA长约、每代GPU HBM翻倍、AI推理需求指数增长——确实有结构性因素。但历史上每个存储"超级周期"都以回归均值结束
PHASE I — EXPLORE

任务定义：存储/HBM节点的真正问题

#1 表面任务 vs 真正任务

表面任务：为存储/HBM节点构建认知代理人报告。

真正任务：回答交易者的核心问题——(1) SK Hynix/MU在涨了300%+之后还有空间吗？(2) HBM供不应求能持续到什么时候？(3) 存储的周期性这次真的被打破了吗？

存储是半导体中周期性最强的子行业。每一个"这次不同"的叙事最终都回归均值。但这次确实有新因素：HBM的die penalty(3-4x)、5年SCA长约、每代GPU的HBM翻倍。

#2 两个代理人的选择逻辑

Kwak Noh-Jung (SK Hynix CEO) = GIANT：全球HBM份额第一(50%+)。M15X $15.1B投资。2026全部订单2025H1即签约。可信度8.5/10。

Sanjay Mehrotra (Micron CEO) = INVESTOR：美国唯一DRAM制造商。从4%到25% HBM份额两年完成。CapEx三月内两次上调($18B→$20B→$25B)。可信度8/10。

Samsung作为对照：非代理人但提供第三方验证——HBM4获NVIDIA最高评分是关键格局变化信号。

注意：两位代理人都是HBM最大供应商=天然看多。卖方偏差是最大的认知风险。

#3 成功/失败标准

成功：报告第一眼回答"HBM供不应求能持续到什么时候"；有2027-2028周期拐点的量化分析；有SK Hynix vs MU vs Samsung的交易时机对比。

失败：变成两个CEO的看多回声室(没有反面论证)；忽略周期性(只讲"这次不同")；没有区分HBM3E/HBM4竞争格局变化。

PHASE II — EXPLORE

素材解码：两位CEO讲了什么故事？

#4 核心叙事：两个CEO的超级周期共识

Kwak的故事(份额龙头)：M15X $15.1B提前4个月量产。2026全部预售。HBM3E涨价20%客户接受。首次年度利润超越Samsung。CES 2026亲自会见25家客户。考虑US ADR上市。"AI内存不是周期性业务，是结构性增长。"

Mehrotra的故事(追赶者)：Q2 FY2026收入beat 19%(+196% YoY)、EPS beat 42%。CapEx三月两升：$18B→$20B→$25B。$100B纽约Megafab破土动工。HBM份额4%→25%两年完成。首签5年SCA长约。"只能满足50-66%客户需求。"

跨代理人交叉信号

#5 信号 vs 噪音

高信号：全年合同锁定(硬)、CapEx方向(真金白银)、HBM4良率对比(竞争格局)、DRAM合约价(量化)、5年SCA(商业模式变化)。

低信号：CEO的定性乐观(卖方偏差)、分析师目标价(后视镜)、"超级周期"标签(每次都有)。

最重要的信号来源：不是两位CEO的话，而是MU Q2 beat 19%却跌5.8%。市场在告诉你：业绩好 + CapEx太大 = 担心周期拐点。当最好的业绩得到最差的反应时，市场在定价未来的担忧。

PHASE III — DIAGNOSE

矛盾与张力：超级周期的裂缝

#6 四个核心张力
张力1："这次不同" vs 历史周期 — 两位CEO都说"不是周期性业务"。结构性因素确实存在(die penalty/SCA/GPU HBM翻倍)。但历史上每个"这次不同"都在周期峰值出现。1990s DRAM、2000s Flash、2017-18 DRAM——每次都有"结构性论据"，每次都回归。
张力2：MU beat 19% + 跌5.8% — 史诗级业绩(+196% YoY)+历史纪录FCF——但$25B CapEx吓坏了市场。这恰恰是超级周期中的典型反应：业绩越好→投资越大→市场越担心拐点。
张力3：Samsung HBM4逆袭 — HBM3E时代SK Hynix绝对主导。但Samsung HBM4获NVIDIA"最高评分"。如果Samsung在Vera Rubin拿到30%+——三强争霸 vs 一家独大 = 定价权分散。对整个行业是好事(更多产能)，但对SK Hynix是威胁。
张力4：Die Penalty的双刃剑 — 1GB HBM消耗3-4x标准DRAM面积。AI吞噬20%全球DRAM产能。这推高了所有内存价格(DRAM合约价+170%)——但也意味着手机/PC/汽车的内存成本暴涨。如果终端厂商反弹抵制→需求端可能出现裂缝。
#7 信息空白
  1. HBM合同的取消/推迟条款——"全部预售"但灵活度如何？2008年教训：预售合同在下行周期被大量取消
  2. HBM4真实良率对比——三家都声称"良率好"但没有独立验证数据
  3. 传统DRAM客户的痛苦程度——+170%价格上涨对手机/PC/汽车OEM的影响有多大？反弹何时来？
  4. 2027-2028新产能的精确时间线——Idaho/M15X满产/Samsung扩产同时上线=精确的供给冲击时间
  5. NVIDIA对三家的真实份额分配——UBS预测SK Hynix 70% Rubin，但NVIDIA自己没说

结论：两位CEO的信息都有强卖方偏差。最关键的缺失是独立的周期拐点分析——需要对比历史周期的产能/价格/库存数据来判断"这次真的不同"的概率。

PHASE IV — DIAGNOSE

时间与反面：如果存储周期回归？

#8 什么在加速/减速？

加速中(看多)：每代GPU HBM翻倍(80GB→192GB→384GB) | 推理需求指数增长 | 5年SCA改变商业模式 | HBM TAM $54.6B→$100B(2028) | DRAM供给仅增16%

减速中(看空)：三家同时大幅扩产 | MU beat but drop 5.8% | DRAM合约价+170%引发反弹 | HBM4三家竞争白热化 | "超级周期"叙事在峰值出现

#9 最强反面论证
HBM不是"新范式"，是历史上最大的存储供给扩张即将撞上周期性需求放缓。
  1. CapEx规模史无前例——SK Hynix $15.1B(M15X) + Micron $25B(包括$100B Megafab) + Samsung同步扩产。三家同时ALL-IN是周期峰值的经典信号
  2. 2027-2028新产能集中上线——Idaho fab/M15X满产/Samsung扩产同时完成。如果此时AI需求增速哪怕从130%降到50%——供给过剩
  3. 每个"这次不同"的共性——1990s PC DRAM说"PC永远增长"、2000s Flash说"手机是结构性需求"、2017 DRAM说"数据中心改变了周期"。每次都有"结构性论据"，每次都回归
  4. Die Penalty引发的反弹——手机/PC内存成本暴涨170%。苹果/三星等终端OEM不可能永远接受。替代技术(CXL内存池化)可能被加速采纳
  5. MU的市场反应是最诚实的信号——beat 19%跌5.8%=机构投资者已经在定价周期拐点。"聪明钱"不看后视镜

概率：30-40%。2026年供不应求几乎确定(合同锁定)。2027-2028是真正的考验。供给过剩出现在2027H2-2028的概率最高。

PHASE V — BUILD

报告结构设计

#10-#11 结构决策与偏差标记

核心原则：报告必须同时呈现超级周期叙事和经典周期回归风险。如果只有一面——就是两个CEO的看多回声室。

卖方偏差校正：两位CEO都是HBM最大供应商。他们说"不是周期性"时，你应该问"如果是周期性呢？"。校正方法：用历史存储周期数据(产能增速/价格变化/库存天数)作为base rate，再看结构性因素能不能改变base rate。

关键洞察：MU beat 19%但跌5.8%——这是所有叙事中最诚实的信号。机构投资者比CEO更早定价周期风险。

PHASE VI — VERIFY

成品审视：诚实的自我诊断

#12 做到了什么/没做到什么

做到了：两代理人行动证据完整(Kwak 7条/Mehrotra 7条)；三角验证表展示供需共识和持续时间分歧；HBM4竞争格局变化被突出；周期风险被诚实呈现。

没做到：缺少历史存储周期的量化对比(1990s/2000s/2017)；CXL替代技术分析不够深；传统DRAM客户反弹的定量模型缺失。

#13-#15 三个认知陷阱
陷阱1：卖方偏差 — 两位CEO都是HBM最大供应商，天然看多需求。"AI内存是结构性增长"这句话从卖方嘴里说出来=打折50%听。从买方(NVIDIA/Google)嘴里说出来才是高信号。
陷阱2：幸存者偏差 — 只看涨了300%+的SK Hynix和MU，不看2022-2023存储暴跌50%+的痛苦。存储行业的beta是双向的——涨的时候最猛，跌的时候也最猛。
陷阱3：锚定偏差 — +196% YoY让人觉得增长天经地义。但196%的base是2023年的存储低谷。从低谷算的增速不代表可持续增速。正确的base rate是正常化后的营收水平。
PHASE VII — BUILD

季度验证日历

#18-#19 预测追踪器

存储节点有独特的高频验证信号：DRAM合约价和现货价(月度甚至周度更新)。这是比财报更快的周期信号。

#20 三个投资机会
机会1：MU Forward P/E 11-12x的盈利爆发窗口。如果Q3 $33.5B兑现→P/E进一步压缩。5.8%回调可能是CapEx消化后的买入机会。风险：2027需求放缓
机会2：SK Hynix ADR上市=估值重估催化剂。当前TTM P/E 17x vs MU 36x(TTM)。如果ADR上市→US投资者涌入→估值差距缩小20-30%
机会3：DRAM合约价作为周期信号。月度DRAM合约价是最灵敏的前瞻信号。合约价同比增速从+170%开始放缓=周期转折的最早预警
#21 监控方法论

每月：DRAM合约价/现货价(DRAMeXchange) | HBM报价变化 | TSMC月度营收(CoWoS利用率代理)

每季度：SK Hynix/MU/Samsung Memory财报 | Hyperscaler CapEx指引 | HBM库存天数

事件驱动：ADR上市公告 | 5年SCA签约 | HBM4良率数据 | NVIDIA新GPU配套HBM规格

PHASE VIII — BUILD

信号传导与标的地图

#22 三条传导链
BULL 链1：超级周期持续 → 存储链
NVDA Rubin出货 → HBM需求暴增 → SK Hynix/MU(最先受益) → TSM(CoWoS封装) → AMAT/TEL(设备) → DRAM合约价↑
信号最先在DRAM现货价和HBM报价出现，最后传到设备商(滞后2-3季度)
BEAR 链2：周期拐点 → 传统崩塌路径
Hyperscaler CapEx削减/推迟 → GPU订单取消 → HBM需求放缓 → 库存堆积 → DRAM合约价暴跌 → SK Hynix/MU/Samsung利润率压缩 → 设备商订单取消
信号最先在Hyperscaler CapEx指引变化出现(每季度财报监控)
WATCH 链3：HBM4竞争重构 → 份额迁移链
Samsung HBM4追平 → SK Hynix Rubin份额从70%降到50% → Samsung存储部门利润修复 → SK Hynix估值溢价缩小
信号最先在NVIDIA供应商公告出现。三家HBM4竞标结果是2026H2最重要的格局信号
#23 时间差套利

最可行的时间差：DRAM合约价是月度数据——合约价同比增速放缓2个月以上=减仓信号。合约价加速上涨=加仓信号。不需要等季报。

#24 "不管谁赢都赚"标的

对比：SK Hynix只有在份额领先持续时赢最多。Samsung只有在HBM4追平时赢(但集团稀释)。AMAT和TSM在任何HBM格局下都受益——因为它们在设备和封装层，不关心谁的HBM。

#25 历史案例
2017-2018 DRAM "超级周期"：所有人说"数据中心改变了周期性"。2018Q4价格暴跌40%+。教训：CEO在周期峰值永远说"这次不同"。但结构性因素(数据中心)确实延长了周期——只是没有消除周期。
MU Q2 beat + 跌5.8% (2026.03)：最新鲜的信号。机构看到$25B CapEx就想起2017——大CapEx=供给过剩的种子。当最好的业绩得到最差的反应，市场在定价未来。
SK Hynix 2023年亏损到2025年翻倍利润：从亏损到利润冠军只用了2年。存储的beta是双向的。涨的时候+345%，跌的时候也可能-50%。
PHASE IX — VERIFY

交叉验证网络 — 最终方法论

#26 存储节点的独特验证方法

存储与其他半导体节点不同——它有最丰富的高频价格信号。DRAM合约价、现货价、HBM报价都是月度甚至周度更新。

这意味着：你不需要等季报就能判断周期方向。DRAM合约价同比增速是最灵敏的前瞻指标。增速开始放缓(从+170%降到+100%再到+50%)=周期减速。增速转负=周期拐点。

但注意：价格信号会比产能信号领先。价格可能在新产能上线前就开始松动(预期效应)。2027H1新产能上线→2026H2价格可能开始松动(领先2-3季度)。

#27 7个Claims的交叉验证结果

最终方法论 — 存储/HBM投资4条规则

规则1：DRAM合约价是最灵敏的周期信号
月度更新。合约价同比增速 > +100% = 加速期。增速开始放缓(+170%→+100%→+50%) = 减速信号。增速转负 = 减仓/空仓。不需要等季报。

规则2：CEO说"这次不同"时信50%
结构性因素(die penalty/SCA/GPU HBM翻倍)确实存在，可能延长周期。但不会消除周期。实用策略：在2026年(合同锁定)高确信度持有。2027H1开始提高警惕(新产能上线)。2027H2看到第一个减速信号立刻减仓。

规则3：MU的市场反应比CEO言论更诚实
beat 19%跌5.8%=机构在定价$25B CapEx的周期风险。当最好的业绩得到最差的反应——信市场不信CEO。反之：如果下次beat后涨=周期恐惧消化完毕=更健康的上涨。

规则4：设备层和封装层是"不管谁赢都赚"
SK Hynix vs Samsung vs Micron的份额博弈不确定。但所有三家扩产都需要AMAT设备和TSM封装。如果你不确定存储周期何时结束——持有AMAT/TSM而不是SK Hynix/MU。
