代工制造 Node 3 — 28轮元思考过程

TSMC垄断 + 地缘风险 + CoWoS瓶颈的三重评估  |  2026-03-20
研究笔记 — 思考过程比结论更有价值

28轮元思考 9个Phase 6个claims交叉验证 3条传导链

🧠 28轮元思考的核心结论

  1. TSMC不是一只股票，是全球AI供应链的"税收站"。NVIDIA赢、AMD赢、Google TPU赢、Apple赢——谁赢TSM都赢。先进制程份额>90%意味着所有道路都经过TSMC
  2. 真正的分析焦点不是"TSMC能不能做出来"(能)，而是"地缘风险怎么定价"。三位代理人(Wei/Chang/Tan)在技术领先上完全共识。分歧全部集中在地缘政治和美国onshoring
  3. CoWoS先进封装正在替代光刻成为新的物理瓶颈。需求100万片/月 vs 产能130K——10倍缺口。这个瓶颈直接决定GPU交付速度
  4. Intel Foundry是"故事vs现实"差距最大的标的。股价涨145%但Foundry亏$10.3B、零外部客户。Chang说"doesn't know how"，Tan说"foundation built"——谁对决定5-10年产业结构
PHASE I — EXPLORE

任务定义：代工节点的真正问题

#1 表面任务 vs 真正任务

表面任务：为代工制造节点构建认知代理人报告。

真正任务：回答三个交易者问题——(1) TSM在AI supercycle中值不值当前估值？(2) 台海风险到底怎么对冲？(3) Intel Foundry是value trap还是转折点？

核心挑战：代工制造是六个节点中垄断最严重的(TSMC 70%总/90%先进)，垄断既是最大护城河也是最大风险源(集中度)。

#2 三个代理人的选择逻辑

C.C. Wei (TSMC CEO) = GIANT：全球半导体制造的实际掌控者。看到所有AI芯片订单。$52-56B CapEx决策。可信度9/10。

Morris Chang (TSMC创始人) = INVESTOR：代工模式的发明者。94岁退休教父，60年经验。不受CEO利益约束。可信度9/10。

Lip-Bu Tan (Intel CEO) = CONTRARIAN：美国本土唯一先进制程能力。VC背景+EDA经验=完全不同的认知框架。可信度6/10。

三角定位：在位垄断者(Wei) + 产业元老(Chang) + 挑战者(Tan)。覆盖运营、战略、竞争三个维度。

#3 成功/失败标准

成功：报告第一眼回答"TSMC的70%垄断在地缘风险下能持续多久"；有CoWoS瓶颈的量化分析；有Wei vs Chang vs Tan在onshoring上的分歧图。

失败：变成"TSMC真厉害"的赞歌(没有风险评估)；低估Intel Foundry(只因现在亏损就否定)；忽略CoWoS(新瓶颈)。

PHASE II — EXPLORE

素材解码：三位代理人讲了什么故事？

#4 核心叙事：三个人看同一个TSMC

Wei的故事(运营者)：$52-56B CapEx创历史新高(+30% YoY)，同时说"I'm also very nervous"。N2如期量产、良率超预期。CoWoS 4倍扩产(35K→130K/月)仍sold out。Arizona $165B投资。2025营收$122.9B(+36%)。6/7预测验证准确。

Chang的故事(战略家)："Globalization is dead"。美国制造成本从"高50%"上修到"可能是两倍"。Intel "doesn't know how to be a foundry"。"Learning is local"——制造能力依赖地域性密集协作。退休后言论更坦率。9/10预测准确。

Tan的故事(挑战者)：$25M个人买入Intel股票(最强skin-in-the-game)。裁员32%断臂求生。18A HVM达成但良率仍低。Foundry亏$10.3B。Nvidia $5B投资=信任信号。CHIPS Act使Intel"半国有化"。股价+145%但基本面弱。

跨代理人交叉信号

#5 三个代理人的共识与分歧

完全共识：TSMC技术不可撼动(2-3年) | AI需求真实 | 地缘风险需要认真对待

核心分歧：美国onshoring可行性(Chang: futile / Wei: dream lives / Tan: foundation built) | Intel能否打破垄断(Chang: never / Tan: given time)

关键洞察：分歧不在技术(TSMC赢)而在政治经济学(谁来分散风险)。这意味着短期(2-3年)TSM论据极强，长期(5-10年)取决于政策和Intel执行力。

PHASE III — DIAGNOSE

矛盾与张力：TSMC垄断的裂缝

#6 四个核心张力
张力1：不可替代 vs 不可对冲 — TSMC 90%先进制程份额=所有AI芯片的共同瓶颈。这既是最大护城河(不可替代性)也是最大风险源(如果台海出事=全球AI停摆)。"Silicon Shield"是叙事不是保险。
张力2：Wei的审慎 vs $52-56B的激进 — "I'm very nervous"和创纪录CapEx同时出现。不矛盾——Wei的审慎是纪律性扩张的信号。但$52-56B如果CapEx周期下行=巨大固定成本。
张力3：Chang vs Wei在Arizona上的分歧 — Chang说"expensive, futile"。Wei说"my dream lives"并投$165B。两人都对——Arizona商业上可能不划算(Chang对)，但地缘政治上不得不做(Wei对)。成本高50-100%被视为"保险费"。
张力4：Intel的"故事 vs 现实" — Tan: 18A达成、Nvidia $5B投资、股价+145%。现实：$10.3B亏损、零外部客户、良率仍低。市场买的是5年后的故事，不是今天的财务报表。
#7 信息空白
  1. 台海军事情景的真实概率——Low但Catastrophic，没有可靠的概率模型
  2. CoWoS良率和真实产能数据——TSMC不公开封装具体数据
  3. Intel 18A的真实良率数字——"improving steadily"是模糊表述
  4. Arizona工厂的真实成本差——Chang说"double"，Wei说"manageable"，真实数字不公开
  5. N2→A16的客户分配优先级——Nvidia vs Apple vs Google谁优先？

结论：代工节点的信息质量在六个节点中最高(TSMC月度营收公开)，但地缘政治和Intel转型是两个高不确定性领域。

PHASE IV — DIAGNOSE

时间与反面：如果TSMC垄断被打破？

#8 什么在加速/减速？

加速中(看多)：先进制程份额64%→70% | AI CapEx $150B+/年 | CoWoS产能4倍仍sold out | N2如期+良率超预期 | $122.9B营收+36%

减速中(看空)：台海地缘风险持续 | Arizona成本高50-100% | Intel 18A逐步逼近 | Samsung正在追赶3nm | CHIPS Act资助竞争者

#9 最强反面论证
TSMC不是"不可替代的堡垒"，是建在地震带上的唯一水库。
  1. 台海是真正的尾部风险——TSMC不是分散的，90%先进产能在台湾。Arizona即使全部建成也仅10-15%总产能。Chang自己不用"Silicon Shield"安慰自己
  2. Intel 18A如果成功——美国政府+$25B补贴+Nvidia $5B投资。Chang说"永远不会"但从未有过这种规模的政府+产业联合推动
  3. $52-56B CapEx的下行风险——如果AI CapEx从$150B骤降到$100B，TSMC的固定成本变成利润杀手
  4. Samsung+Intel联合分散——如果两家各拿到10%先进制程份额，TSMC从90%降到70%。定价权和利润率都会松动
  5. 中国"脱钩"加速——中国市场如果完全自给(SMIC+华虹)，TSMC失去一个$20B+市场

概率：15%。TSMC的技术领先在物理上至少2-3年无法挑战。但地缘风险是不可忽视的"肥尾"。

PHASE V — BUILD

报告结构设计

#10-#11 结构决策与Phase 0偏差

核心原则：报告是TSM/INTC/GFS作为foundry investment的可靠性手册。

Phase 0偏差：(1) 低估了TSMC份额提升速度(64%→70%) (2) 低估CoWoS供需缺口规模(100万片需求vs 8万产能)。两个方向相同：系统性偏保守。

CEO偏差标记：Wei的审慎言论仍服务于$52-56B CapEx的合理化叙事。"I'm nervous"让投资者觉得他在审慎决策——但$52-56B本身就是在赌需求持续爆发。

PHASE VI — VERIFY

成品审视：诚实的自我诊断

#12 做到了什么/没做到什么

做到了：三代理人行动证据完整(Wei 8条/Chang 6条/Tan 8条)；三角验证表展示技术共识+政治分歧；CoWoS作为新瓶颈被量化分析；INTC风险被诚实标注。

没做到：台海风险缺少概率模型(定性>定量)；Arizona真实成本差缺少硬数据；Samsung Foundry作为第三方视角不够独立。

#13-#15 三个认知陷阱
陷阱1：Silicon Shield幻觉 — "台湾太重要不会被攻击"是叙事不是保险。Chang本人不用这个叙事——"geopolitical risk is real"。投资决策不能依赖一个无法验证的地缘假设。
陷阱2：CEO偏差校正 — Wei的"nervous"让人觉得他在审慎——但他同时在花$52-56B。Tan的"foundation built"让人觉得Intel在转型——但Foundry亏$10.3B。两个CEO都有叙事动机。
陷阱3：Chang的结构性悲观 — Chang对Intel/Samsung/US onshoring全面悲观。60年经验给他独特权威——但退休者也可能低估后来者的创新。Tan的VC思维可能找到Chang想不到的路径。
PHASE VII — BUILD

季度验证日历

#18-#19 预测追踪器

代工节点有一个独特优势：TSMC每月10日公布营收。这是整个半导体行业最高频的一手数据。

#20 四个投资机会
机会1：TSM作为"AI税收站"。Forward P/E ~22x在AI supercycle中不贵。月度营收是最高频验证信号。链条：TSMC月度→季度财报→ASML设备订单
机会2：INTC的国策赌注(极高风险/极高回报)。如果18A 2026H2获得外部客户=可能是历史性买点。否则是value trap。等催化剂不预判
机会3：CoWoS瓶颈受益链。TSMC CoWoS扩产拉动AMAT/TEL封装设备。封装设备增速可能超过光刻设备
机会4：地缘恐慌后的TSM买入。每次台海紧张TSM跌5-10%→但TSMC基本面(技术领先/供不应求)不变=系统性买入窗口
#21 监控方法论

每月10日：TSMC月度营收——整个半导体行业最可靠的领先指标

每季度：TSMC季报(CoWoS利用率/CapEx执行) | Intel财报(Foundry亏损/客户公告) | Samsung Foundry(良率/客户流失)

事件驱动：台海军事动态 | Intel Foundry外部客户签约 | CHIPS Act资金拨付 | Arizona里程碑

PHASE VIII — BUILD

信号传导与标的地图

#22 三条传导链
BULL 链1：AI CapEx上行 → 代工链
Hyperscaler CapEx↑ → NVDA/AMD/GOOGL下单 → TSM(制造+封装) → ASML(设备) → AMAT(GAA/封装设备)
信号最先在TSMC月度营收出现，ASML设备订单滞后2-3季度
BEAR 链2：台海风险升级 → 分散链
台海紧张↑ → TSM短期暴跌 → INTC(避风港)短期上涨 → GFS(成熟制程替代)受益 → 但长期所有半导体受损
信号最先在TSM ADR(即时反应)。INTC和GFS的反向走势是对冲工具
WATCH 链3：Intel Foundry转折 → 产业重构链
Intel 18A获外部客户 → TSMC垄断松动预期 → TSM估值天花板下降 → INTC估值重估↑ → 设备商(ASML/AMAT)受益(更多fab)
信号最先在Intel Foundry客户公告。这是5-10年的慢变量
#23 时间差套利

最可行的时间差：每月10日TSMC月度营收公布 → 用它预判TSMC季度财报 → ASML设备订单 → 整条设备链。TSMC月度数据是最强的前瞻信号。

#24 "不管谁赢都赚"标的

对比：INTC只有在Intel转型成功时才赢大。GFS只在成熟制程需求稳定时赢。TSM在垄断持续时赢最多，但即使分散到70%仍是最大赢家。

#25 历史案例
2022 Pelosi访台：TSM当天暴跌——然后3个月内收复全部跌幅。因为基本面(技术领先/供不应求)完全没变。地缘恐慌=买入窗口(有条件：基本面不变)。
TSMC 2025关税危机：Trump威胁100%关税→Wei用$100B追加投资换豁免。看CEO做什么：用投资换政策——这是一种定价权的体现(你不敢真的对我加税)。
Intel Gelsinger被替换：Chang提前预言"像演讲比赛"。Gelsinger被迫离职+Foundry亏$10B验证了Chang的判断。听退休教父的行业判断，可信度极高。
PHASE IX — VERIFY

交叉验证网络 — 最终方法论

#26 代工节点的独特验证优势

代工是六个节点中信息质量最高的——TSMC每月10日公布营收。这意味着：

你每个月都有一次"验证窗口"。不需要等季报。TSMC月度营收 → 预判季度方向 → 预判设备订单方向 → 预判整条供应链方向。

代工节点的信息传导链：TSMC月度营收(Day 10) → TSMC季报(4/7/10/1月) → ASML积压(每季度) → AMAT/LRCX订单(每季度) → 设备安装(6-12月滞后)

#27 6个Claims的交叉验证结果

最终方法论 — 代工制造投资4条规则

规则1：TSMC月度营收是最强领先指标
每月10日公布。月度数据 → 预判季报 → 预判设备订单 → 预判整条链。这是整个半导体行业最高频的一手信号。

规则2：地缘恐慌 + 基本面不变 = 系统性买入机会
台海紧张→TSM暴跌5-10%→但N2如期/CoWoS sold out/营收+30%不变。历史模板：2022 Pelosi事件后3个月收复。条件：基本面确认不变。

规则3：Wei审慎时相信Wei，Tan乐观时打折Tan
Wei 9/10可信度 + CEO利益偏差。Tan 6/10可信度 + 转型期CEO天然过度乐观。Chang 9/10但退休者可能低估后来者。权重：Wei 40% / Chang 35% / Tan 15% / 行业数据 10%。

规则4：TSM是核心持仓，INTC是期权
TSM: AI供应链的"税收站"，Forward P/E ~22x在supercycle中不贵。INTC: 极高风险/极高回报的5年赌注，不适合核心仓位但值得监控催化剂(18A外部客户)。GFS: 成熟制程避风港，低波动。
