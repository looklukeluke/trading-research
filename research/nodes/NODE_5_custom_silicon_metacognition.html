终端自研芯片节点 — 28轮元思考过程

从"每家都在自研"到"Broadcom是隐形赢家"  |  2026-03-20
研究笔记 — 思考过程比结论更有价值

28轮元思考 9个Phase 3代理人交叉验证 5条投资信号

🧠 28轮元思考的核心结论

  1. 自研芯片的真正受益者不是自研者本身，而是Broadcom（设计服务~80%）和TSMC（代工） — "卖铲子给淘金者"是Node 5最清晰的投资逻辑
  2. 三个代理人代表三条路径，但只有Srouji（Apple）有100%交付记录 — Musk 30个预测仅4个准确，Keller还在从零到一。权重不应相等
  3. 最反直觉的发现：每家自研公司仍在大量购买NVIDIA GPU — 自研是supplement不是replace。xAI同时花$18B买GPU又花$20B建TeraFab，说明自研和外购是并行轨道
  4. AVGO是Node 1和Node 5的结构性交叉点 — Jensen对了→网络收入↑；自研加速→ASIC设计收入↑。ASIC增速44.6% vs GPU 16.1%让AVGO的第二条腿越跑越快
  5. 最大风险：自研是永远"即将到来的威胁"，但NVIDIA系统级优势让替代成本永远偏高 — 10年来自研一直在"加速"，但NVIDIA收入也在加速
PHASE I — EXPLORE

任务定义：自研趋势对NVDA估值的影响 + AVGO作为受益者

#1 表面任务 vs 真正任务

表面任务：分析终端自研芯片趋势，展示Musk/Srouji/Keller三人认知。

真正任务：帮一个趋势交易者回答 "自研芯片浪潮对我持有/关注的NVDA、AVGO、TSM意味着什么？该加仓、减仓、还是换标的？"

更准确地说：自研芯片年年说"要替代NVIDIA"，但NVDA年年涨。这次不一样了吗？如果不一样，谁是真正的赢家？

用户的交易系统核心是"选品+周期匹配"。Node 5的任务不是百科全书式罗列谁在做芯片，而是找到自研趋势中的"卖铲子"标的。

#2 V3报告的关键发现

V3报告中Node 5的核心判断已经很清晰："自研芯片的赢家是Broadcom（设计服务~80%）和TSMC（代工），不是自研者本身。"

三个代理人的任务不是推翻这个结论，而是用一手认知证据验证或修正它。如果三人的行动证据都指向"自研加速但仍需Broadcom+TSMC"，这个结论就从"假设"升级为"高置信度判断"。

#3 成功标准 vs 失败标准

成功标准：第一眼就能看到"自研趋势下该买谁、为什么、什么时候验证"；有信号传导链（自研加速→AVGO收入↑）；有验证日历（Baltra、TeraFab、MI400时间节点）。

失败标准：堆砌三人芯片路线图但没有投资逻辑（技术报告）；把Musk的激进时间线当真（fan fiction）；展示自研趋势但不回答"所以买谁"（和事佬）。

PHASE II — EXPLORE

三人叙事：赌垂直整合 vs 证明"先做对再做大" vs 卖IP替代方案

#4 三个代理人到底讲了什么故事？

隐藏叙事解码

核心发现：三人代表三条完全不同的自研路径 — Musk赌规模暴力（花$38B同时买GPU+建fab），Srouji赌精密执行（16年零延迟交付记录），Keller赌生态颠覆（RISC-V+开源替代专有架构）。但他们有一个共同点：全部都需要TSMC/Samsung的代工能力。

跨文件交叉信号

#5 Musk叙事的深层解构

Musk是三人中最复杂的认知节点。他同时做着自相矛盾的事：

  • 嘴上：建TeraFab"不再依赖别人"
  • 手上：$18B买NVIDIA GPU + 继续订购NVIDIA芯片
  • 钱上：$16.5B给Samsung代工 + $20B自建fab

这不是矛盾，是"最大化optionality"策略。Musk不是在选一条路，是在同时铺三条路然后看哪条先通。

但对投资者来说关键问题是：TeraFab成功概率多少？ Tesla零半导体制造经验、需要首次增发融资（自2020年来首次）、2nm制程门槛极高。Electrek的批评很到位："Tesla's TeraFab ambitions ignore its total lack of semiconductor experience."

"Musk的时间线预测准确率约28%（4/14）。技术方向通常正确，但时间线需乘以2-3倍。"
#6 Srouji叙事：唯一有100%交付记录的代理人

Srouji的数据是三人中最干净的：12/12以上产品按时交付，连续16年零重大延迟。

路线图清晰且可验证：A4(2010)→M1(2020)→M5(2025)→C1调制解调器(2025)→Baltra服务器芯片(2026-2027)。

核心框架："颠覆性产品不能用传统芯片来打造"（Disruptive products cannot be built from legacy chips）。这不是空话 — Apple每次自研都让前供应商股价暴跌：

  • 2017年弃Imagination Technologies GPU → 股价当日-70%
  • 2020年弃Intel x86 → Intel持续下跌至今
  • 2025年C1调制解调器 → Qualcomm预测2027年100%被替代

唯一的未知数：Baltra是Apple首次进入数据中心芯片领域。过往消费电子的成功能否外推到服务器？这与Broadcom合作开发 — 又一次证明自研者需要AVGO。

#7 Keller叙事：批评者+IP授权的第三条路

Keller的独特价值不在于Tenstorrent本身（$3.2B估值 vs NVIDIA $3.2T），而在于他同时在Apple、AMD、Tesla、Intel都做过芯片的一手经验。他的判断基于五个场景的亲身参与。

核心论点："Whatever Nvidia does, we'll do the opposite."

这些数字解释了为什么Keller回应Sam Altman的$7T说"我可以用不到$1T做到" — 他认为行业每一层的利润堆叠让最终成本膨胀了7-10x。

关键限制：Keller每家公司都待2-3年。他是"架构天才"不是"商业化执行者"。Tenstorrent是他第一次完整商业化考验。RISC-V 25%全球份额听起来惊人，但几乎全在嵌入式，数据中心份额<1%。

PHASE III — DIAGNOSE

矛盾与张力：自研公司都还在买NVIDIA

#8 四个Node 5的核心矛盾
矛盾1：自研者都还在大量买NVIDIA GPU
xAI: 555,000 GPU ($18B)。Apple: Private Cloud Compute先用M5过渡。Google TPU 7代了仍然买NVIDIA给某些workload。自研是"增加选项"不是"替代供应商"。这对NVDA的看空叙事是最大反驳。
矛盾2：Musk的30个预测只有4个准确 vs 他真金白银投$38B
如果只看言论，Musk可信度3/10。但如果看资金流向（$18B GPU + $16.5B Samsung + $20B TeraFab），他是全球在芯片上投资最多的个人。该信他的话还是他的钱？答案：信钱（方向），不信嘴（时间线）。
矛盾3：ASIC增速44.6% vs NVIDIA收入也在加速
自研叙事常用ASIC增速对比GPU增速来论证"替代"。但这是base effect — ASIC从小基数高增速，GPU从$100B+基数仍保持增长。蛋糕在变大，ASIC切的比例在增加，但GPU的绝对蛋糕也在增加。这不是零和博弈。
矛盾4：Apple最成功的自研者，偏偏对NVIDIA威胁最小
Srouji 100%交付率、A4到M5完美路径。但Apple的自研芯片全在消费电子/edge，不在数据中心训练。Baltra才是第一次进数据中心，而且还是和Broadcom合作。"最有证据的成功案例"恰好不威胁NVIDIA的核心市场。
#9 信息空白
  1. Broadcom自研芯片设计收入中Google TPU占多少比例（是否过度集中？）
  2. Tesla TeraFab的融资来源和时间表（需要增发多少？稀释比例？）
  3. Apple Baltra的性能目标 vs NVIDIA H200/B200 的具体对比
  4. 自研芯片的总拥有成本（TCO）vs NVIDIA GPU的TCO — 没有独立第三方比较
  5. Google/Meta/Amazon内部对自研 vs 外购的资源分配比例变化趋势

结论：公开信息系统性偏向"自研是威胁"叙事。但资金流向（每家都继续买NVIDIA）讲了相反的故事。

#10 代理人可信度权重不应相等

结论：当三人意见冲突时，Srouji的判断权重应为3x Musk。Keller的技术判断可信但商业判断打折。Musk的资金流向可信但言论打折。

PHASE IV — DIAGNOSE

最强反面：自研是永远"即将到来的威胁"

#11 加速 vs 减速
#12 最强反面论证：自研是永远的"下一年"
"自研芯片将替代NVIDIA" — 这句话每年都说，每年NVIDIA的收入都在创新高。
  1. 时间线记录：Google TPU始于2015，Amazon Trainium始于2018，Tesla Dojo始于2021（已失败）。10年过去了，NVIDIA的数据中心收入从$5B涨到$115B+
  2. 系统级优势无法单点突破：自研芯片能做到硅片性能匹配NVIDIA，但CUDA+NVLink+cuDNN+TensorRT+NGC容器生态+数万开发者支持=系统级护城河。替代成本不是"做一颗芯片"的成本，是"重建整个生态"的成本
  3. 自研的隐性成本：Apple自研芯片团队数千人、Srouji领导16年持续投入。Tesla从2016年建团队到HW3量产花了3年、Dojo投入数十亿后失败。自研的ROI被严重高估 — 成功案例只展示收益，不展示前期投入和失败项目
  4. Keller的反驳为什么不成立：他说"利润堆叠可以消除"，但NVIDIA的利润=研发再投入的源泉。73%毛利率支撑每年$12B+研发，形成"高利润→高研发→高性能→高需求→高利润"的飞轮。打破利润堆叠=打破飞轮=需要另一个飞轮替代
  5. Jensen的战略反应速度：$20B收购Groq证明NVIDIA不是静态靶子。竞争者在追赶CUDA，Jensen在加码推理、物理AI、软件平台。追赶者永远在追赶一个也在移动的目标

概率评估：自研在未来5年内真正替代（而非补充）NVIDIA在数据中心的主导地位 — 10-15%。

但自研继续高速增长并成为NVIDIA的结构性利润压缩力量 — 70-80%。

投资含义：自研不是买入AMD/SHORT NVDA的理由。自研是买入AVGO/TSM的理由。

#13 Musk的TeraFab：最激进也最不可预测

TeraFab的5个关键风险：

  • 零经验：Tesla从未运营过半导体工厂。芯片设计≠芯片制造。差距类似于"会开车"和"会造发动机"
  • 资金需求：$20-25B需要Tesla首次增发融资（自2020年来首次），股东稀释不可避免
  • 2nm制程：只有TSMC在3nm有量产经验，Samsung在2nm还在摸索。Tesla从零开始做2nm？
  • 人才：Dojo团队解散时约20名工程师流失至DensityAI。Pete Bannon离职。建fab需要完全不同的人才
  • 时间线：Musk说3月21日"启动" — 但"启动"可能意味着任何事，从"开始选址"到"开始设计"

对比：Intel花了$20B+和3年建Arizona fab（已有60年制造经验），仍然延迟。Samsung Austin fab多次良率问题。Tesla从零开始的TeraFab在2030年前量产2nm芯片的概率 — <20%。

#14 但不要因此低估Musk的方向判断

历史模式：Musk在每个领域都做"疯狂的赌注"，时间线永远延迟，但最终方向通常正确：

  • SpaceX可回收火箭：2015年说"明年量产"，2018年才稳定。但现在是全球唯一
  • Tesla FSD芯片：2016年组团队，2019年HW3量产。3年延迟但证明汽车公司可以自研芯片
  • Starlink：被嘲笑为不可能，现在是最大卫星互联网运营商

结论：TeraFab的时间线不可信，但"终端公司需要掌控芯片供应链"这个方向判断可能是对的。关键不是"TeraFab 2028年能不能量产"，而是"5-10年后Tesla是否有自己的芯片制造能力"。答案大概率是yes — 只不过可能不是通过TeraFab，而是通过收购或深度合作。

#16 最大的结构性遗漏：AVGO的双链交叉点

Node 1的元思考已经发现"AVGO是不管谁赢都赚的标的"。Node 5应该把这个逻辑具体化：

AVGO第一条腿：网络（受益于GPU扩张）
NVIDIA卖更多GPU → 数据中心需要更多网络交换/互连 → AVGO Tomahawk/Jericho系列出货↑
AVGO第二条腿：ASIC设计服务（受益于自研加速）
Google/Meta/Apple/ByteDance自研芯片 → 需要AVGO的设计服务 → AVGO ASIC收入$12.2B (+45% YoY)
Broadcom在自研ASIC设计服务市场占~80%份额。Apple Baltra、Google TPU v6、Meta MTIA都经过Broadcom

交叉点意义：NVDA成功 → 第一条腿加速。NVDA被自研侵蚀 → 第二条腿加速。这是"不管谁赢都赚"的结构性原因，不是巧合。

#17 Keller的位置需要重新定义

在Node 1中Keller作为CONTRARIAN批评NVIDIA定价。在Node 5中他有不同的角色：

Keller是"第三条路"的代言人 — 不自研（像Musk/Srouji），也不买NVIDIA，而是卖设计IP让别人自研。这是ARM模式的RISC-V版本。

如果Keller对了（开源+小芯片取代专有大芯片），受益的不只是Tenstorrent，而是整个"设计工具"层：EDA(SNPS/CDNS)、IP授权(ARM/Tenstorrent)、设计服务(AVGO)。

即使Keller的具体公司不成功，他的方向判断如果正确，AVGO/SNPS/CDNS仍然是受益者。

#22 监控方法论

每月：TSMC月度营收（每月10日）— 自研芯片也需要TSMC代工，产能分配变化是领先指标。

每季度：Broadcom财报（ASIC设计收入增速）| Apple供应商变动 | Tesla财报（芯片相关CapEx和时间线更新）。

事件驱动：TeraFab融资公告 | Baltra tape out | AI5/AI6量产确认 | Tenstorrent大客户签约 | Samsung 2nm良率报告。

关键领先指标：Broadcom每季度的AI/ASIC收入是自研趋势最直接的温度计。如果连续两季度增速>40%，自研加速确认；如果降至<25%，自研可能遇到瓶颈。

#23 四个交易机会
机会1：AVGO "自研加速"主题（持续性）
ASIC设计收入$12.2B+45% YoY。链条：自研公司增加→AVGO设计服务收入↑→月线趋势延续。这是Node 5的最核心交易标的。
机会2：Baltra验证赌注（2026年H2-2027年）
Apple进入数据中心=自研天花板被打破的信号。如果成功→AVGO设计服务+TSMC代工双重受益。
机会3：TeraFab融资事件（2026年Q2-Q3）
Tesla增发融资=短期TSLA股价压力，但长期自研芯片方向确认。对NVDA影响有限（Musk自己说"not replacing NVIDIA"）。
机会4：TSMC产能争夺（慢变量）
Apple vs NVIDIA争夺TSMC 2nm产能。如果Apple拿到更多→NVIDIA受限→AMD/AVGO间接受益。AppleInsider已报道"NVIDIA is squeezing out Apple for TSMC foundry capacity"。
PHASE VIII — BUILD

信号传导与AVGO双链交叉点

#24 三条传导链
BULL 自研加速 → "卖铲子"链
自研公司增加 → AVGO(设计服务80%) → TSM(代工) → SNPS/CDNS(EDA) → ARM(IP授权)
信号最先在Broadcom AI/ASIC季度收入出现。AVGO是整条链的晴雨表
BULL 自研+GPU共存 → 双增长链
自研补充GPU → AVGO网络收入↑(数据中心网络需求不降) + AVGO设计收入↑(自研项目增加)
关键：AVGO两条腿同时受益。这是Node 1和Node 5的结构性交叉点
BEAR 自研真的替代GPU → NVIDIA利润压缩链
ASIC份额超过临界点 → NVDA定价权下降 → 毛利率从73%压缩 → NVDA估值重估
概率10-15%在5年内。但即使发生，AVGO/TSM仍然受益（自研也需要他们）
#25 AVGO双链交叉分析（Node 5核心发现）

结论：在4个场景中，AVGO在3个场景都是正面。唯一的负面是"AI CapEx泡沫破裂"（概率15-20%）。AVGO是Node 5分析中风险回报比最优的标的。

#26 时间差套利

当Broadcom季度财报显示ASIC设计收入加速，但TSMC的自研相关产能收入还没反映 — 这就是时间差窗口。

#27 历史案例
Google TPU (2015至今) — 7代TPU，Anthropic/Claude在上面训练。证明自研可行但Google仍然同时买NVIDIA。自研和外购长期共存。
