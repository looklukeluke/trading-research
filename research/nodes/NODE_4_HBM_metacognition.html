存储/HBM节点 — 28轮元思考过程

从"供不应求共识"到"周期拐点预警"  |  2026-03-20

28轮元思考 9个Phase 2代理人 存储周期分析

🧠 28轮元思考的核心结论

  1. HBM节点的真正问题不是"有没有需求"（三家产能全部预售），而是"supercycle何时结束" — 存储行业历史上每次"这次不一样"都以崩盘结尾
  2. 两位CEO都极度乐观 = echo chamber — Kwak说"晶圆短缺到2030"，Mehrotra说"供不应求到2027+"。没有contrarian voice。Node 1元思考说"所有人都同意时最危险"——这正是HBM当前的状态
  3. Bear case概率30-40%，远高于GPU节点的15-20% — 因为存储有明确的周期历史（2018崩盘、2022-2023崩盘）。三家同时疯狂扩产是经典的周期顶部信号
  4. MU是美国大型科技股2026年唯一上涨（+62%）= 极强相对强度 — 但问题是：这是趋势确认还是最后的疯狂？Forward P/E 11-12x极便宜，但前提是盈利持续爆发
  5. 最有价值的发现：DRAM合约价是最快的领先指标 — 月度跟踪DRAM合约价 → 季度HBM收入确认 → 股价反映（滞后）。2027H2是关键拐点
PHASE I — EXPLORE

任务定义：表面任务 vs 真正任务

#1 表面任务 vs 真正任务

表面任务：升级HBM认知节点报告。

真正任务：帮一个趋势交易者回答 "MU/SK Hynix/Samsung三家谁是最佳HBM标的，以及supercycle何时可能结束？"

更准确地说：MU过去一年涨了350%，SK Hynix涨了345%，我错过了吗？如果没有，谁的risk-reward最好？以及——存储的经典崩盘剧本何时重演？

用户的交易系统核心是"选品+周期匹配"。存储是所有半导体节点中周期性最强的——选对品种但选错周期位置 = 灾难。

#2 代理人偏见结构

两位CEO都是supply-side speaker——天然偏多：

  • Kwak（SK Hynix CEO）：掌控全球50%+ HBM产能，他说"供不应求"是在为$15.1B M15X投资辩护
  • Mehrotra（Micron CEO）：刚把CapEx从$18B追加到$25B，他说"供给紧张到2027+"是在为历史最大资本支出辩护

关键缺失：没有买方（hyperscaler）视角。没有contrarian voice。没有独立的周期分析师。两个卖方CEO说"供不应求"，就像两个房地产开发商说"房价还要涨"——他们可能是对的，但不能只听他们的。

#3 "就是这个" vs "不对"

成功标准：清楚的周期拐点预警指标；三家公司的差异化交易逻辑；可操作的验证日历和领先指标。

失败标准：又一份"HBM供不应求"的共识报告（共识=零信息量）；只有CEO乐观引述没有反面论证（echo chamber放大器）；没有历史周期对比（忽略存储行业最核心的特征）。

PHASE III — DIAGNOSE

矛盾与张力：四个HBM节点没面对的矛盾

#6 四个核心张力
张力1："供不应求"叙事 vs 三家都在疯狂扩产
Kwak: M15X $15.1B，2027满产50K片/月。Mehrotra: CapEx $25B+，Idaho/NY/Singapore同时扩建。Samsung: HBM产能计划+50%。三家合计扩产速度是历史最快的。经典存储周期的剧本：供不应求→疯狂扩产→2-3年后新产能同时上线→供给过剩→价格崩盘。区别只是这次被AI需求延迟了多久。
张力2：两位CEO都极度乐观 vs 没有contrarian声音
Node 1元思考的第5条规则："所有人都同意时最危险"。当前HBM节点的状态：两位CEO都说供不应求到2027+，36/37分析师Strong Buy SK Hynix，31位Strong Buy MU。市场共识太满=任何负面意外的冲击都被放大。MU史诗级业绩后跌5.8%就是预警信号——市场已在寻找"卖出理由"。
张力3：HBM是"结构性需求" vs 存储行业历史上每次supercycle都以崩盘结尾
2016-2018: 加密+手机驱动→DRAM价格翻倍→2019年价格暴跌50%+。2020-2021: WFH+5G驱动→价格飙升→2022-2023价格崩盘。每次行业都说"这次不同"。这次AI确实可能不同（每代GPU配套HBM翻倍+推理需求指数增长），但"不同"不意味着"没有周期"，只意味着"周期被延长"。
张力4：MU在美国股市2026年唯一上涨的大科技股 vs 可能是最后的疯狂
MU 2026 YTD +62%，过去一年+350%。Forward P/E 11-12x极便宜。但便宜的前提是盈利继续爆发（Q3指引$33.5B = +260% YoY）。如果增速从+260%降到+30%，Forward P/E瞬间不再便宜。极强的相对强度可以是趋势确认，也可以是"最后一个灯塔"。
#7 信息空白
  1. 2027-2028新产能上线总规模：三家同时扩产，但没有人给出合计新增晶圆产能的汇总数字。这是判断供需拐点的关键缺失。
  2. Hyperscaler真实HBM库存天数：CEO说"只能满足50-66%需求"，但hyperscaler可能在囤货（就像2021年芯片短缺时的双重下单）。如果库存天数已从30天涨到90天+，"需求"里有多少是真实消耗、多少是库存积累？
  3. HBM4良率：决定2026H2实际产出。三家都声称良率"好于预期"但没有给出数字。TSV 16层堆叠的良率数据是黑箱。
  4. 存储周期历史对比：2018崩盘从peak到trough用了12个月，DRAM价格跌50%+。2022-2023崩盘更快。当前周期的时间轴和幅度对比完全缺失。
  5. AI CapEx的真实ROI：$700B+年投入需要产生多少AI应用收入才能justified？如果2027年ROI不达标→CapEx削减→HBM需求悬崖。

结论：公开信息系统性偏向乐观。两个CEO都在讲"对自己CapEx决策有利的故事"。缺失的是独立的供需平衡表和历史周期对比。

PHASE IV — DIAGNOSE

时间与反面：如果完全错了

#8 加速 vs 减速
#9 最强反面论证
"HBM supercycle是存储周期的变种，不是例外。2027-2028新产能集中上线，如果AI需求增速从+130%降到+30%，HBM价格将暴跌50%+，利润率从58%回归20-30%。这是传统存储的经典剧本，只是时间被AI推迟了2年。"

展开论证：

  1. 供给端：SK Hynix M15X 2027年中满产（+50K片/月）+ Micron Idaho fab 1提前到2027年中 + Samsung产能+50% → 2027H2是供给集中上线的窗口
  2. 需求端：AI CapEx增速不可能永远+130%。从+130%降到+60%，在绝对值上仍是巨额支出，但在"增速"上就是放缓——存储定价对边际增速极敏感
  3. 历史模式：2018年加密崩盘 → GPU库存积压 → 存储需求突然消失 → DRAM价格12个月内跌50%+。当前AI需求结构性更强，但"结构性更强"和"免疫周期"是两回事
  4. 利润率均值回归：SK Hynix 58%营业利润率是存储行业历史最高。存储行业长期平均营业利润率20-25%。即使假设"新常态"30-35%，从58%回归也意味着盈利腰斩
  5. 价格纪律风险：三足鼎立格局下，任何一家为抢份额降价→价格战→全行业利润率崩溃。Samsung历史上多次在追赶时采用"反周期扩产+降价"策略

概率：30-40%。比GPU节点的15-20%高一倍，因为存储有明确的周期历史模式。但如果发生，不是-30%是-60%——存储股的下行beta历来是最大的。

#10 Bear case为什么比GPU节点更高

GPU节点的bear case是"AI CapEx泡沫"（15-20%概率）。HBM节点的bear case除了继承AI CapEx泡沫风险外，还有存储行业特有的周期风险：

  • GPU没有周期历史——NVIDIA作为GPU垄断者从未经历过典型的"供过于求"价格崩盘。存储行业过去20年经历了至少4次。
  • GPU有定价权，存储是商品——NVIDIA毛利率73%因为有CUDA生态锁定。DRAM/HBM本质上是commodity，定价取决于供需平衡，不取决于生态。Micron的5年SCA试图改变这点，但一个合同不能改变行业本质。
  • 三家扩产 vs NVIDIA独家——GPU节点是NVIDIA垄断+AMD追赶，供给端只有一个真正的volume player。HBM节点是三家同时大幅扩产——供给端的竞争烈度完全不同。

所以HBM的bear case概率 = AI CapEx泡沫(15-20%) + 存储周期回归(15-20%) = 30-40%。两个风险可以叠加。

#13 可信度评分的卖方偏见问题

Kwak可信度写成8.5/10，Mehrotra写成8/10。但这两个评分的基础是：

问题：两人的预测准确率高是因为过去2年处于上升周期。在supercycle中，卖方CEO说"需求强劲"永远是对的——直到周期转折的那个季度。这和"到目前为止一切正常"的逻辑一样——从高楼跳下的人在落地前的每一秒都是"正确的"。

缺失的独立验证：没有买方分析师、没有hyperscaler CTO、没有存储周期研究机构的独立视角。两个卖方CEO互相验证=circular validation。

应保留的核心：CapEx数据、产能数字、良率声明、订单锁定状态、价格变动、份额变化。其他都是填充。

#15 最大遗漏
  1. 存储周期历史对比：这是Node 4最独特的分析需求。GPU节点不需要，因为NVIDIA没有周期崩盘历史。但存储节点不做周期对比=最大的分析失职。
  2. 利润率均值回归路径：SK Hynix 58%利润率→行业长期均值20-25%。中间的路径是什么？是平滑回归还是"阶梯式下跌"（每次财报miss一次台阶）？
  3. HBM库存天数跟踪：当前缺失。如果hyperscaler从30天库存涨到90天→说明在囤货→未来需求可能被透支。这是2021年芯片短缺→2022年崩盘的exact pattern。
  4. Samsung的"反周期扩产"历史：Samsung在2008年和2018年都采用过"别人减产我扩产"的策略，用价格战打死竞争对手。如果2027年HBM价格开始松动，Samsung可能再次采用这个策略。
PHASE VII — BUILD

验证日历：什么时候能验证？

#18 Node 4缺失的核心：什么时候能验证supercycle？
#19 监控方法论

月度（最快领先指标）：DRAM合约价变化——这是存储行业最灵敏的温度计。TrendForce月度报价追踪。当合约价从"上涨"变"持平"=第一个预警信号。

季度：SK Hynix财报(1/4/7/10月) | Micron财报(3/6/9/12月) | 关注营业利润率趋势和CapEx指引变化 | HBM库存天数（需从hyperscaler财报推算）。

事件驱动：三家CapEx指引变化（上调=加速，下调=预警） | Hyperscaler AI CapEx指引变化 | Samsung是否启动"反周期扩产"策略 | HBM4良率的第三方验证。

#20 四个关键交易窗口
窗口1：MU Q3 FY2026财报（2026年6月）
$33.5B指引是"史诗级"数字。Beat→趋势延续。Miss→peak earnings确认。DRAM合约价在5月的走势是领先指标。
窗口2：HBM4良率验证（2026年Q3-Q4）
三家都声称HBM4良率"好于预期"。实际量产数据将在Q3-Q4显现。如果良率确实高→ASP更高+产出更多=双重利好。如果良率不达标→供给更紧但利润率承压。
窗口3：Samsung份额争夺（2026年Q4-2027年Q1）
Samsung HBM4获NVIDIA"最高评分"。如果Samsung在Vera Rubin拿到30%+份额→三足鼎立格局确认→长期价格竞争加剧。对SK Hynix是份额侵蚀，对Micron是被进一步挤压。
窗口4：2027H2供需拐点（关键观察期）
SK Hynix M15X满产 + Micron Idaho fab投产 + Samsung +50%产能。如果此时AI CapEx增速已从+130%降到+40-50%→供需拐点出现。DRAM合约价月度变化是最快信号。
#21 历史周期模式参考

模式：每次周期从peak到trough约12-15个月。如果当前supercycle的peak在2027H1（新产能上线前最后一个供不应求季度），trough可能在2028H1-2028H2。但如果AI需求增速持续>产能增速，peak可能被延后到2028-2029。

PHASE VIII — BUILD

信号传导与标的地图

#22 三条传导链
BULL HBM需求持续 → 全内存链受益
HBM需求持续 → MU/SK Hynix盈利持续 → 1GB HBM消耗3-4x晶圆 → 传统DRAM供给被挤压 → DDR5/LPDDR6涨价 → 整个memory链受益
信号最先在DRAM合约价（月度）出现，最后传到存储设备股（滞后1-2季度）
BEAR 2027新产能上线 → 经典存储崩盘
三家新fab同时满产 → HBM供给增速超过需求增速 → HBM价格下跌 → 利润率从58%压向30% → CapEx削减 → 但产能惯性持续12个月 → 更深的价格崩盘
信号最先在DRAM合约价走平/下跌出现。从合约价拐头到股价见顶通常有1-2个月的时间差
WATCH "不管谁赢"受益链
TSMC（HBM先进封装CoWoS/MR-MUF）+ ASM International/Lam Research（HBM设备）无论HBM是SK Hynix主导还是三足鼎立，封装和设备需求都在增长
这些公司不承担HBM价格波动风险，但享受HBM产能扩张的确定性
#23 时间差套利

核心时间差：DRAM合约价是月度数据，但MU/SK Hynix的股价主要由季度财报驱动。当合约价已转跌但下一个季度财报还没发布时——这就是"信息领先期"。反过来也成立：当合约价已加速上涨但股价还没反映时。

#24 "不管谁赢都赚"标的

注意：与GPU节点不同，HBM的"不管谁赢"标的在bear case下保护较弱。GPU节点的AVGO/TSM在"NVIDIA vs 自研"两个方向都受益。但HBM如果整体周期回落，封装和设备需求也会下降——只是滞后1-2季度。所以HBM的"不管谁赢"更准确地说是"不管SK/MU/Samsung谁的份额多"都受益，但不能免疫周期。

#25 历史案例
2018年加密崩盘 → GPU库存 → 存储崩盘
加密需求消失 → GPU库存积压 → NVIDIA砍单 → 存储需求骤降 → DRAM价格12个月内跌50%+。传导链条清晰：终端需求消失 → 中间环节库存积压 → 存储最后受冲击但跌幅最深。
2022-2023存储崩盘
WFH需求结束+手机销量下滑 → DRAM价格15个月内跌40%+ → SK Hynix季度亏损 → 行业大幅削减CapEx → 2024年触底反弹。从亏损到利润率58%只用了18个月——存储的波动性是所有半导体节点中最大的。
当前AI周期 vs 历史的关键区别
历史周期的需求驱动是消费级（手机、PC、加密）——需求弹性高，消费者可以推迟购买。AI周期的需求驱动是企业级CapEx（hyperscaler建数据中心）——一旦规划就有惯性。这意味着AI周期的peak可能更高、持续更久，但trough的深度取决于CapEx惯性消退的速度。
