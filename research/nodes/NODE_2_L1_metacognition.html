§1 节点定义与边界Node Definition & Boundaries

为什么L1协议作为一个节点？

L1协议是区块链技术栈的最底层——提供去中心化信任的基础设施层。没有L1，就没有智能合约、没有DeFi、没有稳定币、没有NFT。所有上层应用都依赖L1的安全性、去中心化和经济模型。L1的边界是清晰的：直接定义共识机制、区块生产和经济模型的协议层。

为什么这样划分？

我将L1协议与其他节点分离，因为：

  • 技术栈位置：L1是"规则制定者"，L2是"规则执行者"，中间件是"信息传递者"。三层分工明确。
  • 估值框架差异：BTC用货币属性估值，ETH用平台估值，TRON用支付网络估值——三种完全不同的逻辑，却都属于L1。
  • 竞争动态：L1之间存在直接竞争（用户、开发者、TVL争夺），而L1与L2是互补关系。

覆盖的子行业：

  • Layer 1区块链：BTC、ETH、SOL、TRON、AVAX、ADA、COSMOS、SUI、APTOS等
  • 共识机制：PoW（BTC）、PoS（ETH 2.0）、DPoS（TRON）、PoH（SOL）等
  • 协议经济模型：代币经济学、gas机制、通缩/通胀设计
  • 跨链协议：LayerZero、IBC——连接不同L1的桥接层（边缘归属）

排除的子行业Excluded Sub-sectors

  • Layer 2：Arbitrum、Optimism、Base、ZK Rollups → 归入Node 7 (DeFi)因为与DeFi生态强绑定
  • 中间件/预言机：Chainlink、The Graph → Node 3
  • 区块浏览器/数据：Etherscan、Dune Analytics → 工具层，非协议
  • staking服务/Liquid Staking：Lido、Rocket Pool → Node 7 (DeFi)

§2 关键公司归类逻辑Company Classification Logic

L1协议节点的归类核心是协议所有权+代币经济模型控制权。我将以下实体归入此节点：

BTC → 无公司实体

中本聪创造了BTC协议，但无公司、无CEO、无董事会。归类依据：BTC是最大的L1，其代币经济学（2100万上限、PoW）是整个行业的基准。

ETH → Ethereum Foundation + Vitalik

非营利组织，但核心开发者有很强影响力。归类依据：ETH是最大的智能合约平台，TVL~$60B，开发者6000+。

SOL → Solana Foundation + Solana Labs

高TPS高性能链，零售端采用快速增长。归类依据：SOL是ETH最强挑战者，开发者增长最快(+80% YoY)。

TRON → TRON DAO + Justin Sun

发展中市场支付通道，3.3亿账户，日均USDT转账$10B+。归类依据：TRON在支付领域的实际采用被严重低估。

AVAX/COSMOS/ADA → 各基金会

边缘包含，归类依据：市值排名靠前但在主报告中作为竞争格局参考，未深入分析。

为什么排除矿机制造商？

矿机生产商（比特大陆、嘉楠耘智）属于Node 1（挖矿与能源）——矿机是能源转换设备，不是L1协议本身。L1协议定义的是链上规则，不是生产这些规则的硬件。

§3 认知偏差Cognitive Biases in Research

研究L1协议时，我识别出以下系统性认知偏差：

CRITICAL
英文来源偏差

英文加密媒体（CoinDesk、CoinTelegraph、The Block）覆盖硅谷项目，忽视发展中市场采用。TRON在尼日利亚$921亿年交易量、巴西90%加密活动是TRON-USDT——这些被西方分析视为笑话，但数据真实存在。

CRITICAL
技术偏见（TPS崇拜）

分析师用TPS比较L1——但BTC最慢（7 TPS）却最大（$1.2T市值）。TPS是可量化指标，但去中心化程度、安全性、开发者生态无法简单量化，导致"快=好"的隐性偏见。

WARNING
市值偏差（Market Cap Bias）

BTC$1.2T、ETH$400B vs TRON$12B——市值大的项目获得更多研究和媒体报道。但这不代表市值小的项目不重要：TRON的3.3亿账户可能比某些$50B市值链更有实际影响。

WARNING
存活者偏差（Survivorship Bias）

我们只研究"活着的"L1（BTC、ETH、SOL）。死掉的L1（EOS、Terra/LUNA、FTM）提供的重要教训被忽视。例如Terra的算法稳定币实验失败是理解稳定币风险的关键。

WARNING
创始人光环效应

Vitalik的透明度 + 高智商形象 = ETH被高估？Justin Sun的争议 + 营销风格 = TRON被低估？创始人个人形象影响了我对项目的客观评估。这是需要警惕的。

CONFIRMED
叙事同质化偏见

BTC"数字黄金"叙事、ETH"世界计算机"叙事——这些是行业叙事而非技术事实。叙事影响价格，但叙事本身是动态的、可被操控的。

§4 与上下游的交叉影响Cross-Node Value Flows

▲ 上游节点

Node 1: 挖矿与能源 — L1协议消耗能源（BTPoW）或锁定资产（PoS质押）。ETH转PoS后电力消耗下降95%+。BTC的PoW消耗决定挖矿成本底线，影响价格支撑。

▼ 下游节点

Node 3: 中间件与预言机 — Chainlink为L1提供外部数据。L1的可靠性决定中间件需求。

Node 4: 交易所与托管 — L1代币在交易所上市流通。托管机构托管L1资产。L1的安全审计结果影响托管合规。

Node 5: 稳定币 — USDT/USDC运行在L1上（ETH、TRON、Solana）。稳定币是L1的主要使用场景之一。

Node 7: DeFi — DeFi TVL ~$200B建立在L1/L2之上。L1的安全性、gas费、可编程性直接决定DeFi的成本和安全性。

⇄ 横向影响

Node 8: 监管与政策 — L1代币是否被定义为证券？SEC对TRON的执法影响整个支付赛道。BTC ETF获批是L1监管合法化的里程碑。

价值传导关键洞察：

L1是整个区块链价值链的"锚点"。BTC ETF获批→机构资金流入→DeFi TVL增长→稳定币需求增加→交易所交易量上升。这是一个自上而下的价值传导链。

§5 信息缺口Information Gaps

研究L1协议时，以下关键数据无法获取或无法验证：

G1: TRON真实活跃用户

3.3亿累计账户 vs DAU？TRON DAO报告的数据缺乏第三方审计。Bot交易比例？有多少是真实用户？

G2: SOL真实用户DAU

SOL日活地址>ETH，但大量是MEV机器人。真实零售用户数无法从链上数据直接获取，需要应用层数据。

G3: L1开发者地理分布

Electric Capital报告开发者总数，但无法知道多少开发者在发展中市场为中国、东南亚、非洲项目工作。

G4: 跨链TVL的真实归属

DeFiLlama的TVL数据包含跨链协议重复计算。某协议在ETH和SOL同时部署，TVL被计算两次。真实L1净TVL无法精确获取。

G5: BTC非流动性供应变化

已知~100万BTC未动，但交易所净流入/流出、机构托管持有量、矿工持币量都是估算值而非精确数据。

G6: ETH L2价值捕获模型

ETH L2的利润如何回流L1？Blob费用是否足够？ARB/OP代币通胀对ETH价值的稀释效应有多严重？

G7: TRON USDD脱锚历史

TRON曾发行算法稳定币USDD，脱锚后如何恢复？Justin Sun对USDD的支持是否可持续？USDD与USDT的关系？

G8: 新L1链的真实去中心化程度

SUI、APTOS、SOL都是高TPS链，但验证者数量、去中心化程度、停机风险缺乏透明报告。

§7 决策日志Decision Log

研究过程中的关键决策记录：

Decision 1: 选择Vitalik、Justin Sun、Satoshi作为认知代理人

理由：三人代表三种完全不同的世界观。Vitalik是理想主义者（去中心化优先），Justin Sun是实用主义者（用户不在乎去中心化），Satoshi是历史参照（BTC的原始设计意图）。

альтернатива：选择"机构投资者"（如Paul Tudor Jones）作为Agent——但他们的公开言论缺乏技术细节。

结果：三视角提供了有价值的张力。Vitalik的透明度和Justin Sun的争议形成鲜明对比。

Decision 2: 将TRON纳入主报告而非边缘提及

理由：330M账户和$10B+日均USDT转账是客观事实，不能因为创始人争议就忽视。西方分析的TRON偏见是研究机会而非排除理由。

альтернатива：只研究BTC、ETH、SOL——最常见的"L1三巨头"叙事。

结果：TRON提供了发展中市场视角，发现了"巴西90%加密活动是TRON-USDT"这一关键洞察。

Decision 3: 用"生态位"框架替代"赢者通吃"叙事

理由：分析BTC、ETH、SOL、TRON在同一维度比较（TPS、TVL）会产生误导。BTC是商品，ETH是平台，TRON是支付通道——不能用同一框架估值。

альтернатива：按照TVL排名或市值排名依次分析——这是大多数加密报告的做法。

结果：生态位框架揭示了"L1不是零和博弈"的核心洞察。

Decision 4: 不深入分析Terra/LUNA崩溃案例

理由：Terra是L1协议，但其崩溃更像稳定币（Node 5）和DeFi（Node 7）的问题而非纯粹L1技术问题。时间有限，选择聚焦。

альтернатива：将Terra作为"L1失败案例"深入分析——但这会模糊节点边界。

结果：保持了节点边界的清晰性，但牺牲了"算法稳定币L1"这一新兴类别的覆盖。

Decision 5: BTC为主报告核心，ETH为次要

理由：BTC市值$1.2T + ETF验证 + 减半周期叙事，是最值得深入研究的L1。ETH竞争压力增大，相对确定性降低。

альтернатива：BTC、ETH、SOL各占1/3篇幅——这是大多数加密报告的均衡做法。

结果：BTC叙事完整，ETH竞争分析到位，TRON提供了差异化视角。
